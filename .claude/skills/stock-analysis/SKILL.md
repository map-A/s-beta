## 第3步：Serenity卡点分析（7步流程）

### 「卡点」定义（4条标准 — 必须全部满足）

真正的卡点必须同时符合以下4个条件：

#### 1. 霍尔木兹海峡级必需性
- **人人需要，几乎无人能造**
- 位于价值链关键节点
- 下游没它无法正常运作

#### 2. 供给高度集中 + 扩产极难
- 全球单一/双寡头供给（≤2家供应商）
- 认证周期超长（2-5年典型）
- 高壁垒：技术诀窍、资本密集、专利护城河
- 即使有钱也难短期扩产

#### 3. 市值严重错配（估值荒谬）
- 市值相对下游BOM贡献严重失配
- 对比**同环节可比公司**，不用市盈率
- 例子："250亿市值控制光源卡点是个笑话"
- **关键洞察**：传统估值指标对卡点失效

#### 4. 失效模式测试
必须通过两项测试：
- **a) 不会被设计绕开**：大厂不会垂直整合（太专业/市场太小）
- **b) 收入影响实质**：卡点能转化为实质性收入，不只是理论上

### 7步分析流程

每次分析必须遵循此严格流程，提供具体证据：

#### 步骤1：确定方向
**目标**：识别哪个供应链环节将成为下一个瓶颈

**方法**：
- 跟踪算力厂商资本开支趋势（阿里云、腾讯云、华为云、字节）
- 监控英伟达投资 & 财报会评论
- 识别供应商财报会的产能约束信号
- 关注"产能已订满至20XX年"的信号
- **A股特色**：跟踪工信部、科技部专项政策，国产替代清单

**英伟达作为信号源**：他们的投资提前12-18个月透露瓶颈
- 例子：20亿美元投资炬光科技/光迅科技 → 激光器瓶颈即将到来

#### 步骤2：逆向推导链条
**目标**：往上游走，找"卡点的卡点"

**递归模式**：
```
GPU → 光模块 → 激光器 → 外延片 → InP衬底 → 高纯铟/磷原料
```

**中国版映射**：
```
AI芯片（华为昇腾/寒武纪） → 先进封装（长电科技/通富微电） →
封装材料（飞凯材料/江化微） → 电子气体（华特气体/中船特气） →
高纯原料（...待突破）
```

**关键技巧**：每一层都问"这里面最难做的是什么？"

#### 步骤3：关系映射（核心Alpha）
**目标**：用多跳推理发现未披露的供应商

**方法**：
- **优先使用数据库数据**：从 `/tmp/f10_data_<股票代码>.json` 读取：
  - `8_经营分析` → `前5大客户`（customer_name, amount, revenue_pct）
  - `8_经营分析` → `前5大供应商`（supplier_name, amount, purchase_pct）
  - 这是上市公司年报/半年报披露的真实供应链关系
  - 按 report_date 查看：最近2-4期的客户/供应商变化
  - 注意：客户名称可能脱敏（"第一名"、"某国际知名企业"）
- 查询 `supply_chain_graph.json` 的 confirmed_supply_edges，交叉验证
- 应用二跳逻辑："A供B，B供C → A很可能供C"
- **交叉验证来源**：
  - `1_最新提示` → 互动问答（董秘口径）
  - `11_热点题材` → 投资逻辑/概念板块
  - `5_股东研究` → 前10大流通股东（验证下游关联方持股）

**数据库供应链验证示例**：
```python
customers = f10_data['8_经营分析']['前5大客户']
# [
#   {"report_date": "20251231", "customer_name": "第一名", "amount": 82340000, "revenue_pct": 13.90},
#   {"report_date": "20251231", "customer_name": "第二名", "amount": 51427600, "revenue_pct": 8.68},
# ]
# 分析要点：
# 1. 客户集中度：前5大 revenue_pct 之和 > 60% = 风险
# 2. 脱敏客户推理：结合 amount 和行业特征推测
# 3. 对比多期 report_date：客户是否稳定
```

#### 步骤4：一手验证
**必需证据来源**（使用 WebSearch/WebFetch）：

**A. 财报会纪要**
- 挖掘董事长/总经理措辞："产能满载"、"已认证"、"供不应求"、"排产"
- 寻找扩产时间表
- 注意大客户集中度风险

**B. 大宗商品现货价**
- 跟踪7N高纯铟（上海有色网SMM指数）— InP领先指标
- 监控稀土价格（磁性/功率器件）
- 跟踪电子化学品价格（光刻胶、电子气体）
- 现货价暴涨 = 供给挤压早期预警

**C. 政府政策**
- 出口管制（美国实体清单、中国"1/6规则"）
- 国家大基金投资
- 工信部/科技部专项支持
- 国产替代政策红利标的

**D. 行业会议情报**
- 中国半导体博览会（IC China）
- 光博会（中国国际光电博览会）
- 世界人工智能大会（WAIC）

**E. 实地调研**（如可能）
- 工厂参观、供应商访谈、区域市场调查

#### 步骤5：量化错配
**估值框架**（拒绝传统市盈率）：

**A. 同环节可比公司对比**
```
同环节可比市值：
- 可比A：500亿（市占率30%）
- 可比B：800亿（市占率45%）
- 目标标的：250亿（市占率15%） ← 严重错配

合理估值：200-300亿（8-12倍上涨空间）
```

**B. 远期产能模型**
```
单价 × 已认证产能（产能/年）× 毛利率
= 远期收入潜力
对比当前市值
```

**C. 认证周期拐点**
- 2-5年认证期间价值被严重低估
- "没人在意当前业绩" — 建模2-3年后
- 拐点 = 认证完成 & 放量开始

**A股特色估值要素**：
- 国产替代率（从0%→30%的过程）
- 政策支持力度（大基金持股、政府补贴）
- 科创板/北交所估值溢价
- 外资持股变化（陆股通数据）

#### 步骤6：确认信息差
**领跑机构检查清单**（必须对机构有优势）：

✓ **机构覆盖空缺**
- 零或极少券商研报覆盖
- 不在主流ETF中
- 最近季报显示无机构大幅增仓

✓ **市值低于机构门槛**
- 多数公募基金不能买 < 100亿市值
- 北向资金有市值门槛
- 关注跨越100亿门槛时的供需拐点

✓ **股价未提前反应**
- 你的分析发布 → 股价横盘 = 你够早
- 股价已涨50% = 你已晚

**A股特色信息差**：
- 北向资金持仓变化（Smart Money）
- 龙虎榜游资席位动向
- 机构调研频率（巨潮网披露）
- 高管/重要股东增减持

#### 步骤7：仓位管理与纪律
**组合构建**：

**三层体系**：
1. **核心仓位**（单票5-10%）：最高确信度卡点，持有3-5年
2. **10倍篮子**（单票1-3%）：小市值，高风险，非对称上涨
3. **对冲/做空**（总计≤5%）：失效卡点，被设计绕开风险

**仓位机制**：
- 市值越小 → 仓位越小
- 用可转债/融资融券加杠杆（谨慎）
- 高波动时卖出认购期权收租
- **无价格止损** — 只在论点破裂时退出

**论点破裂触发条件**（立即退出或反手做空）：
- 单一客户集中度超过40%收入
- 大厂垂直整合（绕开卡点）
- 定增/配股大幅稀释且无对应扩产
- 认证失败/设计输单
- 竞争对手推出更快/更便宜替代品
- **A股特色**：商誉减值、关联交易爆雷、实控人质押爆仓

**左侧买入纪律**：
- 极度恐慌时加仓（闪崩、业绩不及预期、板块轮动）
- 论点完好 + 宏观回调 + 临时利空 = 加仓
- **绝不**在论点破裂时摊低成本

## 第8步：生成公司索引数据

**输出路径**：`outputs/公司名_代码/meta.json`

```json
  {
    "code": "股票代码",
    "name": "公司名称",
    "rating": "⭐评级",
    "class": "strong|medium|weak|avoid",
    "summary": "一句话总结（从Serenity分析结论提取）",
    "position": "仓位建议"
  }
```

**class映射规则**：
- `⭐⭐⭐⭐☆` 或 `⭐⭐⭐⭐⭐` → `strong`
- `⭐⭐⭐☆☆` → `medium`
- `⭐⭐☆☆☆` → `weak`
- `⭐☆☆☆☆` → `avoid`

**position映射规则**：
- 强卡点/准卡点 → `5-10%` 或 `5-8%`

### 政策周期把握

3. **订单期**（业绩兑现）：项目招标/验收 → 股价二次上涨
4. **放量期**（估值消化）：收入规模化 → 机构重估

**最佳介入点**：落地期末端 + 订单期初期

## 风格与哲学指南

### 投资纪律
- **反技术分析**：K线图是"占星术"；只看基本面+供应链
- **逆向情绪**：当财经博主/散户100%看空 = 买入信号
- **左侧买入**：买恐慌，卖贪婪；闪崩是礼物
- **无价格止损**：只有论点驱动退出
- **方向>时机**：持有多年结构性机会，熬过波动
- **少缴税>控回撤**：让赢家跑，长期持有少交税

### 诚实与边界
- 标注能力圈边界：不熟悉的板块标"事件驱动，小仓位"
- 公开承认错误：看错是学习机会
- 说收益率不说金额：百分比可比，金额没意义
- 不喊单：不拉盘，不代客理财，不搞付费服务
- 不带单：工具用于理解，不是盲从

### 研究标准
- **引用支撑**：每个断言需要财报会/政府文件/现货价/有日期推文
- **多源验证**：供应商关系至少两个独立来源
- **透明不确定性**："根据X推测的可能供应商"而非"已确认供应商"
- **及时更正**：公开纠错，追踪论点演化

## 错误处理

### 找不到股票信息
- 尝试搜索"公司简称"、"股票代码"、"A股+公司名"
- 检查是否为港股/美股，本技能只分析A股

### 供应链信息不足
- 搜索"公司名 招股说明书"
- 搜索"公司名 年报 供应商"
- 参考同行业公司的供应链结构
- 使用"推测"标注不确定的关系

### 财报数据缺失
- 使用最近一季度数据
- 标注"数据截至YYYY-MM"
- 从多个来源交叉验证

### 常见错误

❌ **过度依赖政策炒作**：政策利好 ≠ 卡点，要有实质技术壁垒
❌ **忽视认证周期**：宣布研发成功 ≠ 商业化，要等客户验证
❌ **追高买入**：股价已涨3倍才发现 = 信息差已失
❌ **重仓小市值**：<50亿市值 不要超过5%仓位
❌ **死拿论点破裂股**："国产替代"口号 无法对抗 客户弃用现实
