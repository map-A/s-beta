## 1. 核心哲学

Serenity 方法论的起点是一个朴素的观察：每一波产业大浪潮（无论是新能源车、AI算力、还是上一轮的光伏/半导体国产化）里，市场最先涌入的钱，几乎总是追逐"显眼"的终端公司或概念股——整车厂、芯片设计公司、品牌商。但终端公司往往：

- 竞争格局开放，参与者众多，议价权分散；
- 估值已被充分定价，预期早已写进股价；
- 真正决定"这条产业链能不能转起来"的，往往是某个不起眼的中上游环节。

**核心理念**：不炒"xxx概念股"，炒"xxx离不开的东西"。

这句话不限于算力/半导体——它对任何处于扩张期的产业都成立：新能源车离不开的隔膜/六氟磷酸锂，创新药出海离不开的高端中间体/CDMO产能，种业振兴离不开的种质资源和品种权，军工列装放量离不开的特种合金/复合材料，光伏降本离不开的颠覆性设备环节。**方法论的对象是"产业链结构"，行业只是载体。**

## 2. 卡点的本质：跨行业共性特征

无论行业,真正的"卡点"往往具备以下共性结构特征,这是判断"这是不是卡点"的第一层直觉过滤(详细的4条硬标准见第3节):

- **节点稀缺,而非公司稀缺**：卡点是产业链拓扑结构上的稀缺位置,不是某家公司主观营销出来的"独家"。判断方法是看这个**环节**全球有几家能做,而不是看这家公司宣传材料怎么写。
- **替代成本远高于采购成本**：下游宁愿多付钱、排队等货,也不愿意/不能够换供应商。换供应商的代价(认证周期、良率风险、性能损失、监管重新审批)远大于多付的采购金额。
- **扩产周期是"年"而不是"季度"**：新建产能涉及设备交付周期、工艺调试、客户认证、(医药/军工等行业)资质审批,任何环节都不是砸钱能加速太多的。
- **价值量与市值长期不匹配**：因为环节"不起眼"、不在主流叙事里,机构覆盖不足,导致即使产业景气兑现,市值也跟不上。这种错配是收益的来源,而不是风险信号。

## 3. 四项硬标准（通用版 + 多行业案例）

> 这4条标准与 SKILL.md 正文中的定义完全一致,这里补充的是**不同行业下如何具体提问和验证**,帮助把抽象标准落地到任何一个新行业。

### 标准1：霍尔木兹海峡级必需性 —— "人人需要,几乎无人能造"

**通用提问**：如果这个环节明天停产,下游有没有"Plan B"?多久能切换?

| 行业 | 必需性的体现 |
|---|---|
| 半导体/AI算力 | 先进制程光刻机、高端激光器、HBM存储 |
| 新能源车/电池 | 高纯六氟磷酸锂、湿法隔膜涂覆设备、电池级碳酸锂提纯 |
| 创新药/CXO | 关键中间体、特定酶催化工艺、临床前CRO产能(尤其是稀缺动物模型/特殊药理平台) |
| 军工/航空航天 | 高温合金母合金、碳纤维预浸料、特种紧固件 |
| 农业/种业 | 杂交制种技术(尤其是不育系/亲本种子)、特定性状的基因专利 |
| 光伏 | 高纯石英砂(坩埚料)、银粉浆料 |
| 稀土/磁材 | 重稀土分离产能、高性能钐钴/钕铁硼涉及的专利布局 |

### 标准2：供给高度集中 + 扩产极难

**通用提问**：全球/国内能做这件事的有几家?新进入者需要几年才能形成有效产能?

不同行业的"扩产难"成因不同,要分别核实:
- **技术认证型**(半导体设备材料、车规级电池材料、医药原料药):认证周期2-5年,客户不会轻易切换已通过认证的供应商。
- **资质牌照型**(种业的品种审定与生产经营许可、医药的GMP/药品批文、军工的保密资质与定点配套资格):制度性壁垒,纯资本砸不开。
- **资源禀赋型**(稀土分离冶炼指标、特定矿产开采权、高纯石英矿源):受配额/资源储量限制,不是单纯产能问题。
- **工艺诀窍型**(精密光学加工、特种合金冶炼、涂覆/镀膜工艺):隐性know-how,买设备买不来熟练工艺。

### 标准3：市值严重错配

**通用提问**：把这家公司放到它所在的**具体环节**里,跟同环节的可比公司比,而不是跟下游龙头比,贵不贵/便宜不便宜?

注意:不同行业的"可比"维度不同——
- 制造业环节:看单位产能价值量、毛利率水平下的市值/产能比；
- 创新药环节:看管线确定性兑现后的现金流贴现,而不是当期PE(当期利润可能为负或很小)；
- 农业种业:看品种推广面积份额对应的特许权使用费空间；
- 资源类:看储量/配额对应的远期产量 × 价格弹性。

### 标准4：失效模式测试

**a) 不会被设计绕开**:大厂为什么不自己做?常见原因——市场规模太小不值得自建、专业度太高跨界成本高、专利/资质壁垒导致自建即侵权或违规、地理/资源禀赋无法复制。

**b) 收入影响实质**:卡点逻辑必须能在财报里找到对应的收入/毛利率改善证据,不能只停留在"概念图""技术突破新闻稿"层面。**任何行业,"宣布技术突破"与"客户验证通过并下单"之间都有巨大的真空期,这个真空期正是误判最容易发生的地方。**

## 4. 七步流程的方法论原理

> 流程步骤本身不变(详见 SKILL.md 正文),这里解释每一步背后"为什么这样设计",帮助在新行业里灵活应用,而不是机械套用半导体的具体打法。

1. **确定方向**:本质是寻找"下一个产能瓶颈会出现在哪一层"。半导体里看英伟达的Capex信号,在其他行业要换成该行业的"信号源"——新能源车看主流电池厂的扩产公告与车企锁单合同,创新药看头部Biotech的临床进度与CDMO订单饱和度评论,军工看主战装备列装节奏与订货会信息,农业看中央一号文件与种业振兴行动方案的具体表述变化。**关键技巧是找到这个行业里"最先知道扩产计划"的那一类主体,跟踪它的资本开支/订单语言。**

2. **逆向推导链条**:每一层都问"这里面最难做的是什么",递归向上游走,直到碰到一个"几乎所有同行都要用同一家/同一类供应商"的节点。这个技巧是行业无关的,但需要对该行业的工艺流程有基本理解——这也是为什么 Step 2(识别股票信息)里要先做行业基本面调研,再做卡点推导。

3. **关系映射**:用"A供B,B供C → A很可能供C"的二跳逻辑去发现未披露的供应商,核心数据源永远优先用年报/招股书的"前五大客户/供应商"披露——这一条对任何行业都成立,只是不同行业的脱敏方式和披露详细程度不同(消费品行业披露往往更模糊,军工因保密原因披露最少,需要更多用招标公告/中标公示来交叉验证)。

4. **一手验证**:不同行业的"一手证据"类型不同,但都遵循"管理层的具体措辞 + 量价数据 + 政策文件 + 行业会议"四件套逻辑,详见 `verification.md` 的证据等级体系。

5. **量化错配**:核心是拒绝"用下游龙头的估值逻辑给上游卡点定价",而是找到同一环节里的可比对象,详见标准3。

6. **确认信息差**:本质是回答"我比机构知道得早多少"。这一条在小盘股、新兴细分赛道里普遍适用,但要注意:**有些行业(比如军工、部分医药)信息披露天然滞后且不透明,信息差可能来自"愿意花时间啃招标公告/学术文献"这种苦功夫,而不是来自什么内幕优势。**

7. **仓位管理与纪律**:见第5节。

## 5. 仓位管理的底层逻辑

卡点投资的仓位逻辑本质上是在管理**两类不对称风险**:

- **认证失败/被绕开的尾部风险**(下行):一旦发生,基本面逻辑瞬间归零,股价可能腰斩甚至更多。这是为什么"市值越小,仓位越小"——小市值标的的单一客户/单一产品依赖度通常更高,尾部风险更集中。
- **认证周期内市场不关注的耐心成本**(时间损耗):卡点逻辑兑现往往需要2-5年,这期间股价可能横盘甚至下跌,需要足够的仓位纪律才能拿得住。

三层体系(核心仓位/10倍篮子/对冲)的本质是用**确信度分层**来管理这两类风险的暴露度,而不是简单的"重仓我看好的"。

## 6. 常见误判模式（跨行业）

| 误判类型 | 半导体例子 | 其他行业例子 |
|---|---|---|
| 把"概念"当"卡点" | 蹭"国产替代"热点但无实质订单 | 蹭"国产种业振兴"但品种未通过审定;蹭"军工合作"但无具体型号配套 |
| 把"技术突破新闻稿"当"商业化" | 实验室样品当量产能力 | 创新药"临床数据优异"≠获批上市;新材料"小试成功"≠中试放量 |
| 忽视认证/审批周期 | 客户认证2-5年被忽略 | 新药审批被忽略;品种审定/军品定型周期被忽略 |
| 单一大客户依赖 | 单一芯片厂砍单 | 单一军方订单波动;单一车企切换供应商 |
| 把周期性当成长性 | 半导体设备周期顶部 | 农产品价格周期顶部当成永续高景气;大宗商品价格周期顶部 |
| 政策红利期与商业化兑现期混淆 | 补贴期当成盈利能力 | 集采前虚高毛利率当成常态;新能源补贴退坡前业绩当成稳态 |

## 7. 案例库

> 以下案例用于训练"模式识别"能力,具体数据、市值倍数请在分析时用 WebSearch / F10 数据重新核实——产业格局会变化,不代表当前仍然有效。

**正面模式(卡点逻辑兑现)**:
- 半导体设备国产化进程中,部分零部件/材料环节从认证期到放量期出现过数倍估值重估,核心驱动是国产化率从个位数突破到两位数的拐点。
- 动力电池产业链中,部分电解液添加剂、湿法隔膜涂覆设备等"小环节大壁垒"标的,在下游电池厂集中扩产期出现过量价齐升。
- 创新药CDMO环节中,具备特定工艺平台(如多肽固相合成、ADC偶联)的产能稀缺方,在下游创新药管线放量期获得了订单与估值的同步重估。

**反面模式(论点破裂)**:
- 技术宣传与客户验证脱节:某些环节宣布"技术突破"后,因下游客户验证未通过或良率不达标,商业化进度大幅低于预期。
- 单一大客户/单一型号依赖:依赖单一下游大客户的环节,在该客户砍单或切换供应商后业绩大幅下滑。
- 政策窗口期错配:部分标的在政策红利期(补贴、集采前)业绩亮眼,政策退坡后真实盈利能力被证伪。

## 8. 边界：什么时候卡点框架不适用

诚实地标注边界,是这套方法论保持有效性的前提:

- **强品牌/强渠道驱动的行业**(白酒、化妆品、部分消费品):护城河来自品牌心智和渠道网络,而非"供应链稀缺节点",卡点框架对这类行业的解释力有限,更适合用品牌/渠道分析框架。
- **强监管定价的公用事业类**(电力、燃气、部分金融):价值主要由监管定价和资金成本决定,供应链瓶颈不是核心矛盾。
- **纯互联网/平台型公司**:网络效应和用户规模是核心壁垒,供应链概念基本不适用。
- **充分竞争、无明显技术/资质壁垒的大宗制造环节**:即使下游需求旺盛,如果环节本身可以快速新建产能(比如普通钢材、常规化工品),不构成真正的卡点,容易被误判为"概念性卡点"。

当分析对象落入以上边界时,应在输出文档中明确标注"本环节卡点强度有限/不适用本框架",而不是强行套用四项标准给出虚假的高评分。
