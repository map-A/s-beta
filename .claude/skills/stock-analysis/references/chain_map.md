## 0. 通用建模方法：如何为任意行业画产业链图

遇到本文档未覆盖的行业,按以下5步自行建模:

1. **确定终端应用场景**:这个产业链最终服务于谁?(消费者/企业/政府采购)
2. **从终端往上游倒推一层**:终端产品由哪几大模块/材料/服务组成?每一层只拆解到"还能再问一次'这一层最难的是什么'"为止。
3. **标记资本开支与产能周期**:在每一层标注典型的建设周期(月/年)、典型投资额量级,周期最长、投资额最大、参与者最少的那一层,通常就是潜在卡点层。
4. **标记认证/资质壁垒**:在每一层标注是否存在客户认证、政府资质、专利壁垒,壁垒类型决定了"扩产为什么难"。
5. **标记A股概念板块映射**:这一层在A股对应哪个概念/题材板块名称,便于用WebSearch和东方财富/同花顺概念板块工具检索个股。

通用层级命名习惯(不是所有行业都有全部5层,按需裁剪):

```
L0 资源/原材料层  →  L1 基础材料/设备层  →  L2 核心部件/中间体层  →
L3 系统集成/制造层  →  L4 终端应用/品牌层
```

## 1. 半导体 / AI算力硬件

```
L0 资源层:    高纯硅料、稀有气体(氖/氪/氙)、靶材原料金属
L1 设备材料层: 光刻机、刻蚀设备、电子特气、光刻胶、靶材、抛光材料
L2 核心部件层: 光芯片/激光器、外延片(InP/GaAs)、HBM存储、先进封装基板
L3 制造层:     晶圆代工(成熟/先进制程)、先进封装(Chiplet/CoWoS类工艺)
L4 终端层:     GPU/AI芯片设计公司、服务器厂商、云计算厂商
```

**典型卡点定位**:光刻机/高端光刻胶/电子特气/先进封装环节,历史上長期是国产化率最低、认证周期最长的环节。

**A股概念映射**:半导体设备、半导体材料、先进封装、光模块、算力租赁。

**中美对标参考(需用WebSearch核实最新格局)**:
```
美股标的类型      →  对应A股环节(示例,非确定推荐)
EDA/IP            →  国产EDA公司(国产化率仍低)
光刻设备           →  国产光刻机/光刻机零部件公司
先进封装设备/材料   →  封装基板、电子特气公司
晶圆代工           →  晶圆代工厂
```

## 2. 新能源汽车与动力电池

```
L0 资源层:    锂矿/钴矿/镍矿、石墨矿
L1 材料层:    碳酸锂/氢氧化锂提纯、正极前驱体、负极石墨化、隔膜基膜、电解液溶质(六氟磷酸锂)
L2 部件层:    隔膜涂覆、电解液添加剂、铜箔/铝箔(集流体)、电池结构件
L3 制造层:    电芯/电池包制造(动力电池厂)
L4 终端层:    整车厂(新能源车企)、储能系统集成商
```

**典型卡点定位**:湿法隔膜涂覆设备与工艺、高纯电解液添加剂、电池级碳酸锂提纯产能,在下游电池厂集中扩产期容易出现结构性紧缺。

**认证特点**:车规级认证周期通常2-3年以上,且涉及多轮安全测试(针刺、挤压、高低温),切换供应商成本极高。

**A股概念映射**:锂电池、固态电池、储能、隔膜、电解液。

## 3. 光伏与风电

```
L0 资源层:    高纯石英砂(坩埚料)、工业硅
L1 材料层:    多晶硅提纯、银粉浆料、EVA/POE封装胶膜原料
L2 部件层:    硅片、电池片、逆变器核心器件(IGBT模块)
L3 制造层:    组件封装一体化厂商
L4 终端层:    电站开发运营商、户用光伏渠道商
```

**典型卡点定位**:高纯石英砂(坩埚料)受全球矿源高度集中限制;银粉浆料受国产替代进度和银价波动双重影响;逆变器IGBT模块过去存在进口依赖。

**风电补充层级**:
```
L0-L1: 高强度钢/碳纤维叶片材料、稀土永磁(直驱风机用)
L2:    主轴承(大型风电主轴承长期由少数海外厂商主导)、齿轮箱
L3:    整机制造
L4:    风电场开发运营
```

**典型卡点定位(风电)**:大型风电主轴承是历史上公认的"卡脖子"环节之一,国产化进度是核心跟踪指标。

**A股概念映射**:光伏设备、光伏辅材、风电零部件、海上风电。

## 4. 军工与航空航天

```
L0 资源层:    高温合金母合金原料、碳纤维原丝
L1 材料层:    特种钢材、高温合金、碳纤维预浸料、特种橡胶/密封材料
L2 部件层:    精密锻铸件、特种紧固件、航空电连接器、惯性导航元件
L3 制造层:    分系统/部件总装(发动机、航电系统)
L4 终端层:    主机厂(整机/整弹/整船总装)
```

**典型卡点定位**:高温合金(尤其是单晶涡轮叶片用合金)、碳纤维预浸料、特种紧固件,长期受制于工艺诀窍和保密资质双重壁垒。

**验证特殊性**:军工行业披露天然受限,核实供应链关系应更多依赖"保密资质/质量体系认证披露 + 招标采购网公示 + 型号定型节点新闻"组合,而非依赖客户集中度披露(很多军工企业对军品客户做脱密处理)。

**A股概念映射**:军工新材料、航空发动机产业链、卫星互联网、低空经济。

## 5. 创新药与生物医药（含CXO）

```
L0 资源层:    特种化学原料、生物发酵菌种/细胞株
L1 材料层:    关键中间体、高端辅料、培养基
L2 部件层:    特定工艺平台(多肽固相合成、ADC偶联、连续流化学)、关键酶/试剂
L3 制造层:    原料药/制剂生产、CDMO产能(尤其是稀缺产能,如ADC偶联产能)
L4 终端层:    创新药企(管线持有方)、医院/医保终端
```

**典型卡点定位**:具备稀缺工艺平台产能的CDMO环节(例如特定偶联技术、连续流工艺),以及部分依赖进口的高端药用辅料/培养基。

**认证特殊性**:涉及GMP认证、药品注册批文,认证周期长且涉及多国监管(如出海需FDA/EMA认证),"宣布技术平台领先"与"实际拿到商业化订单"之间的真空期是最容易误判的地方。

**特殊风险**:医保集采对仿制药/部分创新药定价的冲击、临床试验失败的尾部风险,需要在论点破裂条件中单独列出。

**A股概念映射**:CXO、创新药、医药外包、ADC、多肽。

## 6. 农业与种业

```
L0 资源层:    种质资源库、野生近缘种基因资源
L1 材料层:    亲本种子(不育系/恢复系)、转基因性状专利
L2 部件层:    制种基地与制种技术(尤其是杂交制种)
L3 制造层:    种子加工与质量检测、品种审定后的规模化繁育
L4 终端层:    种植户/农业集团、下游加工企业(粮食/油脂/饲料)
```

**典型卡点定位**:核心亲本种子(尤其是杂交水稻/玉米的不育系)和具备性状专利的转基因技术,是种业里最接近"卡点"定义的环节——因为品种权和育种周期(通常需要多个生长周期验证)构成天然壁垒。

**验证特殊性**:核实品种权归属应查农业农村部植物新品种权公告和品种审定公告,而不是公司自我宣传;推广面积数据用全国农业技术推广服务中心年度统计交叉验证。

**特殊风险**:农产品价格周期性强,容易把"价格周期顶部"误判为"卡点驱动的持续高景气";政策性补贴/收储价格变化对种植端利润影响巨大。

**A股概念映射**:种业振兴、转基因、生物育种。

## 7. 稀土与磁性材料

```
L0 资源层:    稀土矿(轻稀土/中重稀土)开采指标
L1 材料层:    稀土分离冶炼(尤其是中重稀土分离技术壁垒更高)
L2 部件层:    钕铁硼/钐钴永磁材料制造、磁体表面处理(防腐蚀镀层)
L3 制造层:    高性能磁体加工(用于电机/风电直驱/精密仪器)
L4 终端层:    新能源车电机厂、风电整机厂、消费电子厂商
```

**典型卡点定位**:中重稀土分离冶炼产能受国家开采+冶炼分离总量控制指标双重约束,是结构性稀缺环节;高性能钕铁硼磁体的晶界扩散等专利工艺也构成一定壁垒。

**A股概念映射**:稀土永磁、稀土开采、磁性材料。

## 8. 特种化学品与新材料

```
L0 资源层:    基础化工原料(石油/煤化工route)
L1 材料层:    电子化学品(光刻胶、电子特气、湿电子化学品)、高性能纤维(碳纤维/芳纶)
L2 部件层:    功能性膜材料(锂电隔膜、光学膜)、特种胶粘剂
L3 制造层:    下游制品加工
L4 终端层:    电子/新能源/航空等各应用终端(与上述行业交叉)
```

> 这一层经常作为半导体、新能源、军工等行业的"共性上游"出现,分析时注意识别一家特种化学品公司是否同时服务多个下游行业——这种"卖水人"模式往往比单一下游绑定更稀缺也更安全。

**A股概念映射**:电子化学品、高性能纤维、可降解材料。

## 9. 消费电子 / 智能硬件（"果链"）

```
L0 资源层:    稀有金属(钽/铟等)、高纯铜箔
L1 材料层:    精密结构件原料(铝合金、不锈钢、玻璃/陶瓷基材)
L2 部件层:    精密加工设备(CNC)、声学/光学模组、连接器、被动元件(MLCC)
L3 制造层:    模组组装、整机组装(代工)
L4 终端层:    品牌商(消费电子品牌)
```

**典型特点**:消费电子供应链的"卡点"往往不是单一环节稀缺,而是**良率/精密加工能力的稀缺**——能满足消费电子品牌严苛良率要求的厂商数量有限,这是一种"软卡点"(工艺能力壁垒而非资源垄断),分析时应弱化"标准3:市值错配"中的资源垄断逻辑,强化"工艺良率护城河"的论证。

**A股概念映射**:消费电子、果链、折叠屏、AR/VR硬件。

## 10. 航运 / 港口 / 物流

```
L0 资源层:    船用钢板、船用发动机核心部件
L1 设备层:    船舶设计(双燃料/LNG动力船型)、港机设备(岸桥/龙门吊)
L2 部件层:    船用大功率发动机、船用电子设备(导航/通信)
L3 制造层:    造船厂(分船型细分)
L4 终端层:    航运公司、港口运营商
```

**典型卡点定位**:这是一个**周期属性远强于卡点属性**的行业——运价/造船价格受全球贸易周期影响巨大,真正的"结构性卡点"较少(港机设备、特定船型设计能力是相对少见的例外),分析时应优先用周期分析框架,卡点框架仅作为补充视角。

**A股概念映射**:航运、造船、港口。

## 11. 扩展清单与适用边界

以下行业**不建议**直接套用本框架,或需要大幅调整权重(原因见括号),分析前先参考 `methodology.md` 第8节"边界"判断是否适用:

- 白酒/食品饮料(品牌与渠道驱动,非供应链稀缺驱动)
- 银行/保险/券商(监管定价与资金成本驱动)
- 房地产(政策与信贷周期驱动)
- 游戏/传媒/互联网平台(内容/网络效应驱动,供应链概念基本不适用)
- 大宗钢铁/水泥/玻璃等充分竞争原材料(产能可快速复制,通常不构成真正卡点,除非细分到特种钢材/特种玻璃等子类)

如果用户要求分析以上行业,应在分析开头明确说明"该行业卡点框架适用性有限",并建议结合其他分析框架(品牌力、渠道、周期位置等),而不要强行输出虚高的卡点评分。

遇到本文档未列出的新兴行业(如商业航天、人形机器人、低空经济、合成生物学等交叉/新兴领域),优先使用第0节通用方法现场建模,并在分析文档中注明"该行业产业链结构为现场推导,建议结合更多一手调研验证"。
