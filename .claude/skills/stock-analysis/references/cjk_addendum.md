## 1. 交易机制与板块差异

> 以下规则可能随监管调整变化,分析具体标的前建议用WebSearch核实当前是否仍然有效,不要直接当作永久不变的事实。

| 板块 | 涨跌幅限制(一般情况) | 主要特点 |
|---|---|---|
| 主板(沪/深) | ±10%(ST股±5%) | 上市门槛相对成熟,覆盖各行业龙头 |
| 创业板 | ±20% | 偏重成长型企业,科技/新兴产业集中 |
| 科创板 | ±20%,新股上市前5日不设限 | 偏重"硬科技",对未盈利企业上市有专门通道(适合早期创新药/半导体企业) |
| 北交所 | ±30% | 偏重创新型中小企业,流动性通常弱于其他板块,卡点标的如果在北交所要格外注意成交额是否支持建仓规模 |

交易制度:A股实行T+1交收(当日买入次日才能卖出),融资融券与转融通机制提供了一定的杠杆和做空工具,但融券标的范围有限,大部分小盘卡点股不一定能融券。

## 2. 国产替代/政策周期框架（跨行业版）

国产替代/政策驱动的产业升级,不仅发生在半导体——种业振兴、高端医疗器械、特种纤维材料、工业软件、农机装备等领域都有类似的政策驱动节奏。通用的四阶段框架:

```
阶段1 概念期(0%→10%替代率): 政策文件发布,题材炒作领先于业绩,
       估值容易脱离基本面,此阶段适合小仓位埋伏、不适合重仓
阶段2 验证期(10%→30%替代率): 部分企业通过客户认证/获得批文/中标,
       业绩开始兑现但市场关注度仍不足 — 这是Serenity方法论建议的最佳介入区间
阶段3 放量期(30%→60%替代率): 收入规模化增长,主升浪,机构开始集中调研和加仓
阶段4 成熟期(60%→80%+替代率): 增速放缓,估值随ROE/成长性回归常态,
       此时新进入者很难再获得超额收益
```

**不同行业对应的政策文件/清单类型(核实时按行业去对应部委查找最新文件)**:

| 行业 | 对应政策文件/清单类型 |
|---|---|
| 半导体/工业软件 | 工信部"卡脖子"技术清单、国家大基金投资名单 |
| 创新药/医疗器械 | 国家药监局优先审评审批名单、国家医保谈判目录 |
| 种业 | 中央一号文件种业振兴相关表述、农业农村部生物育种产业化试点 |
| 军工 | 国防科工局"十四五"装备发展规划相关表述(具体型号信息通常不公开) |
| 新能源/光伏 | 国家能源局新增装机规划、发改委产业指导目录 |

## 3. 资金面与信息差数据源

跨行业通用的资金/信息面数据源(具体平台和披露规则可能调整,核实最新可用渠道):

- **北向资金(陆股通)持仓变化**:反映外资/机构对个股的边际态度,持续净买入通常被视为认可信号,但要注意北向资金可能涵盖被动指数基金的被动调仓,不完全代表主动判断。
- **龙虎榜**:反映短线游资动向,游资进驻通常是短期情绪驱动,不直接等同于产业逻辑认可,卡点投资者应区分"游资炒作"和"机构调研后建仓"两类资金性质。
- **机构调研记录(巨潮资讯网披露)**:跟踪机构调研频率和参与机构类型(公募/私募/外资),调研频率突然上升往往是机构开始关注的早期信号。
- **限售解禁日历**:解禁规模较大且解禁方为财务投资人(而非产业资本)时,短期抛压风险上升,分析持仓时机应避开解禁高峰。
- **股东人数变化**:股东人数持续减少(筹码集中)通常被视为存在主力资金吸筹,但需结合基本面交叉验证,避免单独依赖该指标。

## 4. A股特有风险大全（跨行业版）

以下风险类型不限于任何单一行业,分析任何标的的"论点破裂条件"时都应逐项核查:

- **商誉减值风险**:并购重组后商誉占净资产比例过高(经验阈值约50%以上需高度警惕),业绩承诺到期后未达标的并购标的存在集中减值风险。常见于跨界并购热点行业(如此前的游戏、医药外包跨界并购)。
- **股权质押风险**:实控人/大股东质押比例过高(经验阈值约80%以上需警惕),股价下跌时容易触发平仓连锁反应,加剧股价波动。
- **关联交易风险**:核实"主要客户/供应商"是否为关联方,关联交易占比异常的"供应链合作"需要打折评估其真实性和可持续性。
- **业绩对赌/承诺到期风险**:并购重组业绩承诺到期前一年业绩异常亮眼、到期后大幅下滑甚至商誉减值,是A股并购重组类公司的常见模式,需要核实业绩承诺到期时间表。
- **再融资/定增稀释风险**:大规模定增/配股如果没有对应的产能扩张或业绩增长支撑,会构成股本和每股收益的稀释,需要核实募投项目是否与卡点逻辑一致(募资是用于扩大卡点产能,还是用于偿还债务/补充流动资金)。
- **财务造假/审计风险**:关注审计机构是否变更频繁、是否被出具非标准审计意见、应收账款/存货增速是否远超收入增速等财务勾稽异常信号。
- **ST/退市风险**:核实公司是否存在被实施风险警示(ST/*ST)的可能性(连续亏损、净资产为负、审计意见异常等触发条件),退市风险标的不适合作为卡点逻辑的长期持仓对象,即使短期题材炒作活跃。
- **行业特有合规风险**(按行业补充):
  - 医药行业:临床试验数据造假、医保骗保、带量采购降价风险
  - 农业行业:转基因品种未经审批违规推广、农产品质量安全事件
  - 军工行业:保密资质被取消、型号定型推迟或取消
  - 资源类行业:安全生产事故导致停产整改、环保督察导致限产

## 5. 板块轮动与情绪周期

A股市场具有较强的板块轮动和题材炒作特征,卡点投资者需要区分"产业逻辑驱动的长期机会"和"纯题材轮动的短期炒作",两者在盘面上初期表现可能相似(放量上涨、龙虎榜活跃),但持续性和回撤后的恢复能力差异巨大:

- **产业逻辑驱动**:通常伴随基本面数据(订单、产能利用率、客户认证进度)的持续验证,回调后只要逻辑未破裂,后续仍能创新高。
- **纯题材轮动**:缺乏基本面验证,资金退潮后往往一蹶不振,识别方法是核查上涨期间是否有对应的公告/财报数据支撑,还是单纯跟随板块情绪。

实践建议:在生成最终投资建议时,明确区分当前驱动力是"产业逻辑兑现期"还是"题材炒作期",并在风险提示中注明,帮助使用者管理预期和仓位纪律。
