**关键特征**：

- **产品端**：以"铜价+加工费"定价，本质是来料加工模式
- **客户端**：通过一级供应商（线束厂商）进入整车厂供应链

## 核心竞争力评估

### 优势
1. **A股首家汽车线缆上市公司**：品牌先发优势，资本市场融资能力
2. **客户覆盖广泛**：进入全球主要整车厂供应链体系
3. **全国布局+出海**：七大国内基地+墨西哥工厂
4. **新能源转型**：高压线缆单车价值量提升（传统300-600元 → 新能源800-1500元）
5. **ISO标准参与**：参与起草国际汽车线缆标准

### 劣势
1. **毛利率极低**：6.64%~10%，反映产品大宗商品属性
2. **缺乏定价权**：铜价+加工费模式，利润空间受上下游挤压
3. **产能利用率低**：新能源线缆仅15%，大量产能闲置
4. **负债率高**：69.13%，多基地扩产加重财务负担
5. **新业务无实质**：铜缆高速连接、机器人、航天均在概念阶段

## 投资逻辑与风险总结

### 核心投资逻辑（市场叙事）

1. **铜缆高速连接概念**：NVIDIA GB200 NVL72架构采用铜缆连接 → 卡倍亿设立智联子公司 → 市场预期受益
2. **新能源汽车渗透率提升**：单车线缆价值量从600元提升至1500元 → 卡倍亿作为国内龙头受益
3. **人形机器人+商业航天**：概念布局 → 市场给予估值溢价

### 真实基本面

1. 汽车线缆业务稳定增长（+10.87%），但利润率极薄（毛利率6.64%）
2. 2025年全年扣非净利润为负（-1647.66万元）
3. 新能源线缆产能严重过剩（利用率15%）
4. 铜缆高速连接业务实质贡献<1%
5. 机器人/航天子公司无业绩、无产品

### 风险提示

1. **概念退潮风险**：估值溢价建立在未验证的新业务上
2. **客户集中风险**：第一大客户占28%，前五大占61%
3. **产能过剩风险**：新能源线缆利用率仅15%
4. **铜价波动风险**：虽然不直接影响利润，但影响收入波动
5. **海外扩张风险**：墨西哥工厂投资大，北美市场不确定性高
