## 供应链关键洞察

### 1. 产业链位置

寒武纪处于**半导体/AI算力硬件产业链的L4终端层（AI芯片设计）**，属于Fabless模式——只做设计，制造完全外包。

```
L0 资源层：高纯硅料、稀有气体
L1 设备材料层：光刻机、刻蚀设备、电子特气、光刻胶
L2 核心部件层：HBM存储、先进封装基板
L3 制造层：晶圆代工（中芯国际）、先进封装（长电/通富微电）
L4 终端层：← 寒武纪在这里（AI芯片设计）
L5 应用层：服务器厂商、云计算厂商、AI应用公司
```

**关键判断**：寒武纪不是产业链中的"卡点"供应商，而是卡点的"用户"。它需要采购上游代工厂的产能、使用上游EDA工具、依赖上游HBM存储——这些上游环节才是真正的卡点。

### 2. 竞争格局

**国内AI芯片设计竞争格局（2026年）**：
- **华为昇腾**：市占率70-80%（inferred级），生态最成熟，信创采购首选
- **海光信息**：市占率10-15%（inferred级），x86兼容优势，商业市场渗透率较高
- **寒武纪**：市占率5-10%（推测级），纯AI芯片设计，思元690量产中
- **阿里平头哥**：自研PPU出货"已超寒武纪"（媒体报道，inferred级），但主要服务阿里云
- **其他**：壁仞科技、沐曦、摩尔线程等新兴竞品

**竞争态势**：AI芯片设计层竞争者众多且持续增加，不存在寡头格局。寒武纪的相对优势在于产品线覆盖云边端+较早进入市场，但面临的最大竞争压力来自华为昇腾（信创龙头）和大厂自研（阿里平头哥等）。

### 3. 上下游议价能力

**对上游议价能力：弱** ❌
- 晶圆代工完全依赖中芯国际等少数代工厂，无议价权
- EDA工具高度依赖Synopsys/Cadence/华大九天
- HBM存储受供应商（SK海力士、三星）供给约束
- Fabless模式的天然劣势：不控制产能

**对下游议价能力：弱** ❌
- 前5大客户合计占营收88.66%，客户集中度极高
- 客户（互联网大厂/云厂商）普遍采取多供应商策略，随时可切换
- 大厂具备自研能力（华为、阿里已自研），买方议价权极强
- 信创采购受政策驱动，定价受政府采购预算约束

**综合判断**：上下游均无议价权，这是典型的Fabless设计公司的结构性弱点，也是Serenity框架不看好此类标的的核心原因之一。

## 风险因素

### 供应链风险
- **晶圆代工集中风险**：第一大供应商占采购55.34%，中芯国际产能紧张时寒武纪可能被"插队"
- **先进制程获取风险**：受美国制裁影响，获取7nm以下先进制程代工能力有限
- **HBM存储供给风险**：高性能AI芯片对HBM有刚性需求，海外供应商受出口管制约束

### 需求侧风险
- **大厂自研替代**：阿里平头哥已大规模出货，其他大厂自研趋势不可逆
- **客户集中度**：前5客户占89%，任一客户流失影响巨大
- **政策依赖**：信创采购是重要收入来源，政策变化直接影响业绩

### 其他风险
- **估值泡沫**：PE 349倍，PB 73.6倍，股价创历史新高
- **竞争加剧**：华为、海光、平头哥等强势竞品持续扩大市占率
- **技术迭代**：AI芯片架构快速迭代（如Transformer Engine、稀疏计算等），产品生命周期短
- **人才流失**：芯片设计行业人才竞争激烈，核心团队稳定性是关键

## 结论

寒武纪是国内AI芯片设计赛道的头部企业之一，受益于AI算力爆发和国产替代双重驱动，2025年首次实现全年盈利，2026Q1业绩爆发式增长。但从供应链角度看：

1. **不是卡点，是卡点用户**：寒武纪处于产业链终端L4层，真正的卡点在上游（代工/封装/设备/材料）
2. **上下游均无议价权**：Fabless模式+高客户集中度=两头受压
3. **竞争格局开放**：5-6家竞品+大厂自研，不存在寡头垄断
4. **估值严重高估**：9,471亿市值/349倍PE已将所有乐观预期定价

从Serenity「卡点」方法论的角度，寒武纪不适合作为卡点投资标的。如看好AI算力赛道，应将注意力转向寒武纪的上游——晶圆代工、先进封装、半导体设备/材料等真正的卡点环节。
