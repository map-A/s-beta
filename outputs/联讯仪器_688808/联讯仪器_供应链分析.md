## 供应链关键洞察

### 1. 产业链位置

联讯仪器处于**半导体/AI算力硬件产业链L1设备/仪器层**，具体为：

```
L4 AI服务器/云计算      →  亚马逊AWS、微软Azure、Google
L3 数据中心网络设备     →  思科、Arista
L2 光模块/收发器        →  中际旭创、新易盛、Lumentum ← 联讯直接客户
L1 测试仪器设备 [此处]  →  联讯仪器（688808）
L1 激光器芯片/光芯片    →  源杰科技、长光华芯
L0 原材料/零部件        →  精密机械、电子元器件
```

**SiC产业链位置**：
```
L4 新能源汽车整车        →  比亚迪、特斯拉
L3 电驱动系统/逆变器     →  汇川技术、英威腾
L2 SiC功率模块           →  英飞凌中国、斯达半导、比亚迪半导体
L1 SiC器件测试 [此处]    →  联讯仪器（WLBI测试 43.6%国内份额）
L0 SiC晶圆衬底           →  天科合达、天岳先进
```

### 2. 竞争格局

**光通信测试仪器**：外资强势主导（Keysight+Anritsu约84%），联讯9.9%全市场（#3），国内唯一进前五的本土企业——处于**"追赶者"地位**，依靠国产替代需求快速提升份额。

**光电子器件测试设备**：外资为主，联讯5.2%国内第一——**"小而强"地位**，专注芯片前端工序测试。

**SiC WLBI测试**：联讯43.6%**国内寡头**，SPEA/Aehr为主要竞争者，国内市场外资退出后空间巨大——这是最接近"卡点"的子赛道。

### 3. 上下游议价能力

**对上游**：采购分散，单一供应商占比不超过8%，议价能力**较强**。

**对下游**：中际旭创、新易盛、Lumentum等大客户采购联讯测试仪器，有一定的替代选项（Keysight）——但实际切换成本高（认证周期6-18个月）。随着国产替代加速和联讯技术持续升级，议价能力**逐步增强**。

**定价权体现**：毛利率66.76%（2026Q1），远高于一般制造业，接近优质软件/仪器公司水平——体现了较强的定价权和技术壁垒。

## 风险因素

### 供应链风险
- **进口元器件风险**：艾睿电子采购的核心电子元器件中可能包含部分高端进口芯片（高速ADC/DAC等），若受出口管制限制，将影响设备生产
- **精密机械加工集中**：苏州本地供应链集中，自然灾害/疫情等导致供应链中断风险

### 需求侧风险
- **光模块景气度周期**：AI算力投资若出现周期性回落（云厂商Capex缩减），光模块需求下滑直接影响测试仪器采购
- **CPO量产延迟**：CPO架构若推迟大规模商业化，对应的高端测试需求可能推迟2-3年
- **Keysight重返**：若中美关系改善，出口管制放松，Keysight/Anritsu重新以更有竞争力的条件进入中国市场

### 其他风险
- **估值泡沫**：当前PE 864x、PB 230x是极端泡沫，市场情绪逆转时下跌幅度难以预测
- **大股东减持**：上市6个月解禁后，VC机构集中减持风险
- **技术迭代**：测试仪器迭代快（400G→1.6T→3.2T），若联讯未来研发进度慢于外资对手，市场份额可能逆转

## 结论

苏州联讯仪器是**真实的AI光互联+SiC国产替代受益标的**，核心逻辑清晰：中国光模块厂和SiC器件厂需要高端国产测试设备，联讯是当前最强的国内供应商。业务增长数据真实（Q1 2026收入+142%，净利润+515%），下游客户（中际旭创/新易盛/Lumentum）均属光通信和AI算力产业链核心玩家。

**但供应链分析的结论止步于此**：这是一个好生意，但在当前市值（2361亿）和估值（PE 864x）下，**不是Serenity定义的"卡点投资机会"**——因为卡点投资的核心是在市场认知尚未完全定价时买入，而联讯已经是A股历史级别的热门标的，所有公开信息已被充分（甚至过度）定价。

**等待合理估值区间（600-1200亿）出现时，重新评估是否建仓。**
