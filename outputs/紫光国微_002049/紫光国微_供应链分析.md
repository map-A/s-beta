## 供应链关键洞察

### 1. 产业链位置

紫光国微处于半导体产业链的**L2核心部件层**，拥有三大业务板块：

```
L0 资源层  → L1 设备层  → L2 核心部件层  → L3 系统层  → L4 终端层
硅片/晶圆    EDA/IP/工艺   特种FPGA/SoC    军用电子系统   国防装备
石英材料     特种封装材料   安全芯片(SIM)    安全模组      通信/金融
                            石英晶振        汽车域控      汽车/IoT
                            ← 紫光国微
```

核心竞争力：
- **特种芯片龙头**：国内最早从事特种IC设计的企业之一，FPGA/SoC/DSP/ASIC全覆盖
- **安全芯片全球前列**：SIM卡/金融IC卡芯片全球市占率前茅
- **专利壁垒**：421项发明专利+221项实用新型
- **资质壁垒**：军工资质+银联/国密/CC EAL6+等顶级安全认证+车规AEC-Q100/ISO26262
- **纵向延伸**：收购Linxens（法国微连接器）+自建高可靠封测产线

### 2. 竞争格局

**特种集成电路**：
- 国内主要竞争者：成都华微（688709）、复旦微电（688385）、景嘉微（300474）等
- 紫光国微产品线最全（FPGA+SoC+DSP+ASIC+存储+配套），系统解决方案能力最强
- 竞争格局相对稳定（军工资质壁垒限制新进入者）

**智能安全芯片**：
- 全球竞争者：恩智浦（NXP）、英飞凌（Infineon）等国际巨头
- 紫光国微在SIM卡/金融IC卡芯片全球市占率前列
- 国内竞争者有限，安全认证壁垒高

### 3. 上下游议价能力

**对上游**：⚠️ 议价能力中等
- 前5大供应商占采购50.01%，对晶圆代工伙伴依赖度中等
- 自建封测产线逐步提升自主可控能力

**对下游**：✅ 议价能力强
- 特种芯片：军方客户依赖度高（资质壁垒限制供应商选择）
- 安全芯片：产品通过多项顶级安全认证，客户切换成本高
- 前5大客户合计45.19%，分散度合理

## 风险因素

### 供应链风险
- **Fabless模式**：不控制晶圆制造产能，受代工厂产能分配影响
- **特种工艺依赖**：部分特种芯片工艺对特定代工厂有依赖

### 需求侧风险
- **军工订单季节性**：上半年订单少、下半年集中，导致业绩大幅季节波动
- **安全芯片价格竞争**：国产替代后可能面临更多国内竞争者

### 其他风险
- **紫光系治理**：实控人为紫光集团旗下，集团重整后的长期治理安排需关注
- **收购整合风险**：瑞能半导体+Linxens的整合效果待验证
- **军工信息不透明**：特种芯片业务信息披露有限，投资者难以跟踪订单进展
