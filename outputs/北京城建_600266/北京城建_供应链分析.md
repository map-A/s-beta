## 供应链关键洞察

### 1. 产业链位置

北京城建处于房地产开发行业的**中游——开发运营环节**：
```
土地资源 → [北京城建] → 商品房销售 → 终端购房者
建材/施工 → [北京城建] → 物业管理服务
```

这不是一个具有供应链层级结构（L0→L4）的行业。房地产的核心竞争力在于：
1. **土地获取能力**（招拍挂、城市更新、旧改）
2. **资金成本与融资能力**（国企背景优势）
3. **产品力与品牌**（非卡点逻辑）
4. **政策敏感度**（限购限贷政策变化）

### 2. 竞争格局

房地产开发行业**极度分散**：
- 全国开发商超9万家
- CR10不超过15%
- 北京城建综合实力排名全国第37位（2025年）
- 在北京市场有一定品牌认知度，但非绝对龙头

### 3. 上下游议价能力

**对上游议价能力**：弱
- 建材为标准化大宗商品，价格受市场波动影响
- 关联采购占比过高（68.89%），独立议价能力存疑

**对下游议价能力**：弱
- 房价受政策调控和市场供需影响
- 2025年行业下行期，去化压力大

## 风险因素

### 供应链风险
- **关联交易风险**：关联方采购占68.89%，可能存在利益输送
- **施工质量风险**：过度依赖集团内部施工单位，缺乏充分市场竞争
- **建材价格波动**：大宗商品涨价压缩利润空间

### 需求侧风险
- **政策调控风险**：限购限贷政策反复
- **购买力下降**：居民收入预期弱，购房意愿低迷
- **人口结构变化**：城镇化率放缓，增量需求递减
- **区域集中风险**：公司高度依赖北京市场

### 其他风险
- **信用风险**：2025年全年亏损11.78亿，信用评级可能承压
- **现金流风险**：经营性现金流持续为负
- **资产减值风险**：存货（土地+在建工程）可能进一步减值
- **机构撤离风险**：机构持仓从147家骤降至6家（2026Q1）

---

## 结论

北京城建是一家典型的**区域性国资房企**，其供应链结构简单且高度依赖关联交易。从供应链分析角度看：

1. **无供应瓶颈**：上游建材/施工为完全竞争市场
2. **无下游锁定**：购房者有大量替代选择
3. **关联交易占比异常高**：68.89%采购来自控股股东体系，需关注治理风险
4. **行业处于深度调整期**：2025年全年亏损，Q1虽扭亏但持续性待验证

**该标的不适合作为供应链卡点标的**。房地产行业的核心分析维度应为政策周期、信贷环境、土储质量和信用风险，而非供应链稀缺性。
