## 供应链关键洞察

### 1. 产业链位置

```
L0 射频材料(LCP/MPI等) → L1 射频零部件(天线/连接器/滤波器) → L2 模组(无线充电/声学) → L3 终端设备(手机/卫星/汽车)
```

信维通信的独特价值在于**"材料→零部件→模组"垂直整合**+**多赛道布局**（消费电子+卫星通信+汽车+MLCC）。

### 2. 竞争格局

射频零部件行业**竞争充分**，不存在Serenity定义的卡点。但公司在细分领域有竞争优势：
- 天线：全球龙头之一
- SpaceX连接器：认证壁垒高
- 高端MLCC（参股）：国产替代卡脖子方向

### 3. 上下游议价能力

**对上游**：强（前5大供应商仅占13.79%）
**对下游**：中等（客户A占20.14%，前5大50.03%）
