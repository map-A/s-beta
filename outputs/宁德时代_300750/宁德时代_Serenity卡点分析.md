宁德时代是**全球动力电池龙头**，以45.54%市占率形成寡头垄断，掌握电池系统集成核心技术，是新能源车产业链L3关键组件供应商。

## Serenity「卡点」4项标准评估

### ✓ 1. 必需性（霍尔木兹海峡测试）— **通过**

**为什么是必需**：
1. **所有电动车都需要动力电池** — 核心动力来源，无可替代
2. **系统集成复杂度高** — 电芯+BMS+热管理+结构设计，需要多年技术积累
3. **客户高度依赖** — 理想/小鹏/蔚来100%依赖宁德时代供应

**判断**: 动力电池是新能源车的"心脏"，属于霍尔木兹海峡级必需组件。

### ✓ 2. 供给集中度 & 扩产难度 — **通过**

**市占率**（2026年3月中国市场）:
- 宁德时代: **45.54%**（接近垄断）
- 比亚迪: 17.83%（主要自用）
- LG新能源: 10%+
- 中创新航: 5-8%
- **结论**: 准寡头地位，前2家占63%

**认证周期**: 2-3年
- 产品验证: 6-12个月
- 量产爬坡: 12-24个月
- 总计: **2-3年**（中等偏长）

**壁垒分析**:
- **技术壁垒**: ★★★★☆（高）
  - 麒麟电池: 600Wh/L体积能量密度，行业领先
  - 神行电池: 6分27秒充电至98%，超越比亚迪刀片二代
  - BMS算法: SOC/SOH估算精度，热管理优化
- **资本壁垒**: ★★★★☆（高）
  - 10GWh产线投资: 30-50亿
  - 研发投入: 2025年184.69亿，占收入4.7%
- **专利壁垒**: ★★★★☆（高）
  - CTP技术（Cell to Pack）
  - 麒麟电池结构专利
  - 钠离子电池技术储备

**判断**: 供给高度集中，扩产周期2-3年，**符合卡点标准**。

**为什么不会绕开**:

3. **规模经济显著**: 宁德时代单位成本比小厂低20-30%
4. **专注核心业务**: 车厂更愿意聚焦整车设计，电池外包

**b) 收入影响实质**: ★★★★★（极高）

- 整车成本: 20-30万
- **占比**: **30-40%**（最大单一成本项）

**收入实质性验证**:

- 毛利率: 25-28%（行业领先）
- 客户覆盖: 全球前10大车企中8家

**判断**: 
- 整车厂短期内无法绕开宁德时代
- 收入规模巨大，占下游BOM 30-40%
- **失效模式测试: 完全通过**

- 2026年锂价回升至15万/吨
- 成本上升，毛利率承压
- 但宁德时代议价能力强，可向下游传导

**2. 比亚迪竞争加剧**（★★☆☆☆ 中偏低）
- 比亚迪刀片二代充电速度落后
- 外供规模小（主要自用）
- 短期威胁有限

**3. 技术路线颠覆风险**（★★☆☆☆ 低）
- 固态电池: 宁德时代同步研发
- 钠离子电池: 宁德时代领先
- 氢燃料电池: 商业化遥远

**4. 客户集中度风险**（★★★☆☆ 中）
- 前五大客户占比可能>50%
- 但全球布局分散风险

**5. 海外政策风险**（★★★☆☆ 中）
- 欧美本土保护主义
- 需本地化生产（已布局）

### 退出触发条件

**论点破裂信号**:
1. **市占率跌破35%** → 寡头地位动摇
2. **单一客户占比>40%** → 过度依赖风险
3. **比亚迪/LG推出革命性新电池** → 技术代差逆转
4. **固态电池突破被竞争对手抢先** → 失去技术领先
