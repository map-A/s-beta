## 供应链关键洞察

### 1. 产业链位置

云南锗业位于两条产业链的交叉点：

**锗资源产业链**：
```
L0 锗矿资源层:  ★ 云南锗业自有矿山 + 全球锗矿资源
L1 冶炼层:      锗锭/二氧化锗冶炼（公司核心业务）
L2 深加工层:    锗单晶/光学元件/光纤材料（公司延伸业务）
L3 终端应用层:  红外/光纤/光伏/半导体终端
```

**InP半导体材料产业链**：
```
L0 原料层:      高纯铟 + 高纯磷（外购）
L1 衬底层:      ★ InP衬底（公司新业务，规划45万片4英寸）
L2 外延/器件层: InP激光器/探测器/VCSEL
L3 模组层:      光模块（800G/1.6T）
L4 终端层:      AI数据中心/5G基站
```

**核心判断**：公司处于锗资源产业链L0-L1（有卡点特征）和InP产业链L1（有全球寡头特征但产能极小），两条链的交叉点赋予了独特的稀缺性，但当前估值已严重透支这一稀缺性。

### 2. 竞争格局

**锗产品市场**：
- 国内参与者：云南锗业（最大）、驰宏锌锗（综合冶炼最大）、南京锗厂、韶关冶炼厂等
- 中国控制全球60-70%锗产量，出口管制保护国内市场
- 云南锗业优势：自有矿山+完整产业链（开采→冶炼→深加工）

**InP衬底市场**：
- 全球3-4家供应商：AXTI（美国，中国有产线）、住友电工（日本）、JX金属（日本）
- 云南锗业是国内InP衬底规模化先行者（云南鑫耀半导体）
- 规划产能：45万片/年（4英寸），投资1.89亿元
- 相比AXTI等全球龙头，产能规模仍很小

### 3. 上下游议价能力

- **对上游议价**：中等偏强。自有矿山降低了原料依赖，但InP原料全部外购
- **对下游议价**：中等。锗产品有一定稀缺性溢价（出口管制），但InP衬底尚未形成规模优势
- **整体定位**：锗资源端具有资源禀赋型卡点特征（有限的锗矿+国家出口管制），InP端具有技术型卡点潜力（全球仅3-4家供应商）

## 风险因素

### 供应链风险
- **锗价波动**：锗价受出口管制政策影响极大，政策放松可能导致价格大跌
- **InP原料外购依赖**：InP/GaAs衬底所需原辅料全部外购，无上游自给能力
- **矿山资源有限**：锗矿储量有限，长期面临资源枯竭风险

### 需求侧风险
- **军品业务进展缓慢**：公司自述军品和砷化镓投产进展缓慢
- **InP市场需求不确定性**：AI算力对InP衬底的需求规模和时点存在不确定性
- **替代技术风险**：硅光子技术可能在部分场景替代InP

### 其他风险
- **估值泡沫**：PE 4459倍/股价创历史新高/散户26.7万人，典型概念炒作后期
- **InP项目执行**：1.89亿投资规模相对较小，技术能否突破存在不确定性
- **民营企业家风险**：实控人家族控制，公司治理依赖管理层能力

---

## 结论

云南锗业拥有独特的"锗资源+InP半导体材料"双轮驱动的产业链位置，在锗资源端具有资源禀赋型卡点特征，在InP衬底端具有全球寡头特征的技术卡点潜力。然而：

1. **锗业务**：受出口管制和锗价驱动，本质是资源周期股
2. **InP业务**：具有真正的卡点潜力，但产能极小、尚未贡献利润
3. **估值**：850亿市值/PE 4459倍严重脱离基本面，当前价格已将远期利好充分透支

建议关注InP衬底产能投产进度和客户验证情况，在估值回归合理区间后重新评估投资价值。
