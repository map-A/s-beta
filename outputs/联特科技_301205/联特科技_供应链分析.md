## 供应链关键洞察

### 1. 产业链位置

联特科技位于光通信产业链的**L3制造层（模块集成）**：
```
L0 InP原料 → L1 InP外延片 → L2 激光器芯片(EML/VCSEL)
                                ↓
              L3 光模块集成 ← ★ 联特科技 ★
                                ↓
              L4 AI服务器/云厂商
```

这是一个**集成环节**，核心工作是：
1. 采购上游激光器芯片、TIA/Driver IC等核心器件
2. 通过光耦合工艺将芯片集成到模块中
3. 进行测试和封装
4. 出货给下游云厂商/设备商

**关键洞察**：模块集成环节的壁垒在于**工艺良率**和**客户认证**，而非不可替代的技术垄断。公司掌握的BOX/TO-CAN/COB器件设计和光耦合工艺有一定技术门槛，但不是"无人能造"的稀缺能力。

### 2. 竞争格局

光模块行业呈**金字塔型**竞争格局：

```
Tier 1（全球龙头，市占率>10%）：
  中际旭创(~30%+) → 产品线最全，规模效应最强
  Coherent/Lumentum → 自研芯片+模块垂直整合

Tier 2（强势挑战者，市占率3-10%）：
  新易盛 → 快速崛起，性价比策略
  光迅科技 → 自研芯片能力
  海信宽带 → 背靠海信集团

Tier 3（二三线厂商，市占率<3%）：
  ★ 联特科技 ★ → 海外产能优势，正在突破高端客户
  华工正源、AOI等
```

联特科技处于Tier 3，正在向Tier 2跃迁的过程中。关键看1.6T产品能否如期放量、北美头部客户能否突破。

### 3. 上下游议价能力

**对上游议价能力**：弱
- 核心器件（激光器芯片）供给集中，价格由上游主导
- 公司正在推进关键物料本地化以降低成本依赖

**对下游议价能力**：弱
- 云厂商采购量大，议价能力强
- 行业价格竞争激烈，800G光模块价格持续下行
- 前5大客户集中度65.44%，大客户依赖明显

## 风险因素

### 供应链风险
- **上游芯片供给集中**：激光器芯片全球供给集中在Coherent/Lumentum等少数厂商，地缘政治可能影响供应
- **供应商集中度**：供应商1占采购15.4%，供应链韧性需关注
- **原材料价格波动**：InP衬底、光学组件等价格受全球供需影响

### 需求侧风险
- **客户集中度**：前5大客户占65.44%，大客户订单波动直接影响业绩
- **AI投资周期性**：云厂商Capex存在波动，可能影响光模块采购节奏
- **竞争加剧**：龙头中际旭创规模效应持续扩大，新易盛快速扩张，联特科技面临两头挤压

### 其他风险
- **估值风险**：PE 500x远高于行业均值（行业PE 63.8x），估值泡沫显著
- **技术路线风险**：CPO技术演进可能改变传统可插拔模块市场格局
- **汇率风险**：海外收入占比89%，人民币升值压缩利润
- **境内业务亏损**：境内毛利率为负，盈利完全依赖海外市场

## 结论

联特科技是一家处于**快速成长期的二三线光模块集成商**，受益于AI算力对高速光互联的爆发式需求。从供应链分析角度看：

1. **行业赛道正确**：光模块是AI时代的核心基础设施之一
2. **但自身非卡点**：模块集成环节竞争激烈、供给分散、扩产门槛有限
3. **真正的卡点在上游**：激光器芯片（EML/VCSEL）和InP衬底才是真正"人人需要、几乎无人能造"的瓶颈环节
4. **估值严重偏高**：437亿市值/1亿净利=500x PE，即使按最乐观预测也已透支
5. **短期业绩承压**：Q1 2026营收-9.56%、净利润-83.71%

从卡点投资逻辑出发，应关注联特科技上游的真正瓶颈标的，而非联特科技本身。若从成长投资角度看，需等待更合理的估值和业绩验证。
