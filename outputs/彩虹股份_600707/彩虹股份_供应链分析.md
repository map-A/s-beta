## 供应链关键洞察

### 1. 产业链位置

彩虹股份位于平板显示产业链的上游核心材料层（L1），是LCD面板制造的"地基"供应商。公司同时涉及下游面板制造（L2），形成了"基板+面板"双主业结构。

**产业链图谱**:
```
L0 原材料        L1 基板玻璃         L2 面板制造         L3 模组/终端
┌──────────┐   ┌──────────────┐   ┌──────────────┐   ┌──────────┐
│ 高纯石英砂 │──→│ 彩虹股份     │──→│ 京东方(BOE)  │──→│ 电视/手机 │
│ 硼酸/硼砂  │   │ (G8.5+基板) │   │ TCL华星     │   │ 笔记本   │
│ 铂铑贵金属 │   ├──────────────┤   │ 惠科(HKC)   │   │ 车载显示 │
│ 熔炉设备   │   │ 彩虹光电     │──→│ (LCD面板)   │   │ 工控     │
└──────────┘   │ (8.6代面板)  │   └──────────────┘   └──────────┘
                └──────────────┘
```

### 2. 竞争格局

**基板玻璃全球格局**:
- **康宁(Corning)**: 全球份额~50%+，拥有溢流下拉法核心专利，技术最先进
- **AGC(旭硝子)**: 全球份额~15-20%
- **NEG(日本电气硝子)**: 全球份额~10-15%
- **彩虹股份**: 国内唯一G8.5+量产，全球份额约3-5%（inferred）
- **东旭光电**: 已退出（债务危机）

**面板制造竞争格局**:
- 京东方(BOE): 全球LCD面板份额~30%
- TCL华星(CSOT): 全球LCD面板份额~20%
- 惠科(HKC): 全球LCD面板份额~10%
- 彩虹光电: 全球LCD面板份额约3-5%（inferred）

**关键洞察**: 中国面板产业已实现"下游面板制造领先全球（70%+份额）"但"上游基板玻璃严重依赖进口（自给率约10-20%）"的结构性失衡——这正是彩虹股份的战略价值所在。

### 3. 上下游议价能力

**对上游议价**: 弱-中
- 原材料（石英砂、硼酸等）为大宗商品，价格受市场供需决定
- 铂铑贵金属为稀缺材料，供应集中度较高
- 浮法工艺设备相对成熟，设备供应商选择较多

**对下游议价**: 弱
- 前5大客户集中度71.76%，最大客户占比28.4%
- 面板厂（京东方、TCL华星）体量远大于彩虹股份，议价权在买方
- 基板玻璃国产替代率仍低（10-20%），面板厂可以随时切换回进口供应商
- 康宁作为全球龙头掌握定价权，彩虹股份跟随定价

## 风险因素

### 供应链风险
- **原材料价格波动**: 铂铑贵金属价格波动直接影响成本端
- **技术追赶风险**: 浮法工艺与康宁溢流下拉法在品质上仍有差距，可能限制高端客户拓展
- **合营依赖**: 与康宁的合营关系（虹宁公司）说明高端工艺仍受制于人

### 需求侧风险
- **面板周期下行**: 面板价格具有强周期性，下行期直接影响彩虹光电的利润
- **OLED替代**: 中高端手机、IT产品向OLED转移，减少LCD基板玻璃需求
- **面板厂资本开支放缓**: 面板厂扩产放缓将直接影响基板玻璃采购量

### 其他风险
- **大股东高比例质押**: 咸阳金控质押率68.65%，咸阳城投质押率65.05%
- **关联交易**: 公司与控股股东、合营企业之间存在较多关联交易
- **国企治理**: 国企背景在战略使命上有优势，但治理效率和市场化程度可能不如民企

---

## 结论

彩虹股份的核心供应链价值在于其作为国内唯一G8.5+基板玻璃量产企业的战略稀缺性。在中国面板产业占全球70%+但上游基板玻璃自给率仅10-20%的结构性失衡下，公司承担着国家战略层面的国产替代使命。

然而，从供应链角度看，公司面临的挑战同样明显：（1）下游客户高度集中且议价权在买方；（2）技术路线（浮法）与全球龙头（康宁溢流下拉法）有差距；（3）双主业结构中面板业务的周期性波动掩盖了基板玻璃业务的战略价值。

未来关键观察指标：基板玻璃产能利用率、国产替代率提升速度、面板价格周期拐点、以及公司是否进一步剥离面板业务聚焦基板玻璃。
