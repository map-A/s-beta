## 供应链关键洞察

### 1. 产业链位置

中科飞测处于半导体设备产业链的**核心设备环节**：
```
光学元件/精密机械/电子元器件 → 【中科飞测: 检测/量测设备】→ 晶圆厂良率管理 → 芯片制造
```

这一位置的关键特征：
- **不可替代**：每一步工艺后都需要检测/量测确认良率
- **技术密集**：深紫外光学+AI算法+精密机械的多学科交叉
- **国产替代刚需**：出口管制下国内晶圆厂必须建立国产供应链
- **高毛利率**：47.77%毛利率体现技术溢价

### 2. 竞争格局

**全球检测/量测设备市场（2025年，190.9亿美元）**：
- **KLA垄断**：全球市占率>50%，技术领先3-5年
- **前5大海外企业合计>70%**：KLA+AMAT+以色列三家
- **国产替代率极低**：<10%（推测）
- **中科飞测定位**：国内龙头，对标KLA的国产替代者

**中科飞测vs KLA技术对比**（inferred）：
| 技术维度 | KLA | 中科飞测 |

| 深紫外光学检测 | 全球领先 | 国内领先，差距缩小中 |
| 电子束检测 | 成熟 | 正在验证/放量 |

## 风险因素

### 供应链风险
- **核心零部件进口依赖**：部分高端光学元件可能依赖进口（inferred）
- **技术追赶不确定性**：与KLA在先进制程上的差距需要时间弥合

### 需求侧风险

- **出口管制政策变化**：如果管制放松，国产替代紧迫性降低

### 其他风险
- **估值极高**：PE 20572x/PB 22.92x/股价远超券商目标价
- **机构减仓**：从655家减至130家

## 结论

中科飞测是国内**半导体检测/量测设备**赛道的绝对龙头，对标全球巨头KLA。在美国出口管制背景下，国产替代逻辑非常坚实。

**核心判断**：
1. **检测/量测设备是半导体制造的"卡点"**：KLA全球垄断，国产替代是刚需
2. **中科飞测是国内唯一全品类布局者**：光学+电子束+X光一站式
3. **业绩已进入放量期**：2025年扭亏为盈，营收+48.75%
4. **但股价已严重透支**：332元远超券商目标价223元，机构大幅减仓
