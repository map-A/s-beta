## 供应链关键洞察

### 1. 产业链位置

金风科技位于风电产业链的整机制造层（L3），向上整合叶片、发电机、变流器等零部件，组装成风电整机后销售给风电场开发商。

**产业链图谱**:
```
L0 资源层      L1 材料层           L2 部件层           L3 整机    L4 终端
┌──────────┐  ┌──────────────┐  ┌──────────────┐  ┌────────┐  ┌──────┐
│ 稀土矿   │→│ 钕铁硼永磁   │→│ 发电机/变流器│→│ 金风科技│→│ 风电场│
│ 铁矿    │  │ 碳纤维      │  │ 叶片       │  │ (整机  │  │ 开发商│
│ 石英砂   │  │ 高强度钢    │  │ 主轴承     │  │ 集成)  │  │      │
└──────────┘  └──────────────┘  │ 塔筒       │  └────────┘  └──────┘
                                └──────────────┘
```

### 2. 竞争格局

**国内风电整机格局**（inferred，基于行业公开信息）：
- 金风科技: ~20-30%新增装机市占率（公司自称80%需核实，实际按GW口径约20-30%）
- 远景能源: ~20-25%
- 明阳智能: ~15-20%
- 运达股份: ~8-10%
- 三一重能: ~5-8%
- 其他: 华锐风电、东汽、Vestas中国等

**全球格局**: Vestas、Siemens Gamesa、GE Vernova、金风科技为主要竞争者

**关键洞察**: 风电整机行业集中度适中（CR5约70-80%），竞争格局健康但非垄断。金风科技是龙头但非唯一选择。

### 3. 上下游议价能力

**对上游议价**: 中等
- 核心零部件（叶片/主轴承/发电机）有3-5家供应商可选
- 稀土永磁受稀土价格波动影响（大宗商品属性）

**对下游议价**: 弱-中
- 客户极度分散（前5大仅占29.27%），但风电场开发商多为国企，议价权较强
- 应收账款339亿元（占营收46.5%）——下游话语权强的体现

## 风险因素

### 供应链风险
- **稀土价格波动**: 永磁直驱技术路线依赖钕铁硼，稀土价格波动影响成本
- **大型主轴承供应**: 全球仅少数供应商（SKF/舍弗勒/NSK等），是潜在瓶颈

### 需求侧风险
- **行业周期性**: 风电装机受政策周期和电网消纳能力影响
- **价格竞争**: 整机价格持续下降，毛利率承压
- **应收账款风险**: 339亿元应收账款，风电场开发商（多为国企）付款周期长

### 其他风险
- **负债率71.1%**: 风电场投资开发需要大量资本开支
- **无实际控制人**: 公司无控股股东和实际控制人，治理结构存在不确定性
- **海外地缘风险**: 土耳其、摩洛哥等新兴市场存在汇率和政治风险

## 结论

金风科技是中国风电整机绝对龙头，正经历量利齐升的经营拐点。从供应链角度看，公司处于风电产业链的系统集成层（L3），向上整合零部件、向下对接风电场开发商。行业竞争格局健康但非垄断（5-6家主要竞争者），公司凭借规模优势和永磁直驱技术路线保持领先地位。

但关键结论是：**风电整机不是卡点环节**——真正的卡点可能在上游部件（大型风电主轴承、稀土永磁材料），这些环节才具备"供给集中度高、扩产难"的特征。
