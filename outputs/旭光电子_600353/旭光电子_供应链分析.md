## 供应链关键洞察

### 1. 产业链位置

旭光电子横跨三条产业链：

```
电力设备链：L0 金属原料 → L1 真空灭弧室/开关管 → L2 电力开关设备 → L3 电网
军工链：L0 电子元器件 → L1 军工电子模块 → L2 武器系统 → L3 国防
电子材料链：L0 氮化铝粉体 → L1 【旭光电子-氮化铝陶瓷/基板】 → L2 半导体封装 → L3 AI芯片
```

### 2. 竞争格局

**真空灭弧室**：宝光股份为直接竞争对手，行业双寡头格局（国内）
**氮化铝陶瓷**：日本企业（京瓷等）主导全球市场，国内企业追赶中，旭光电子是国内具备全产业链能力的少数企业之一
**核聚变电子管**：细分领域参与者极少，但市场极度不确定

### 3. 上下游议价能力

**对上游**：中等——氮化铝粉体部分依赖进口，议价权有限
**对下游**：中等偏弱——客户分散（前5大仅占11.15%），单个客户议价权有限

## 结论

旭光电子是"电力设备+军工+电子材料"三位一体布局的公司，其中氮化铝陶瓷在AI散热领域有一定准卡点潜力，但公司整体：

1. 传统电力设备业务（真空灭弧室）竞争格局清晰但非垄断
2. 氮化铝业务收入占比仅5.6%，尚未成为核心支柱
3. 核聚变概念高度不确定
4. PE 245倍严重高估

从供应链角度，氮化铝陶瓷是值得长期跟踪的准卡点方向，但旭光电子本身需要等待氮化铝业务占比和盈利能力实质性提升后再重新评估。
