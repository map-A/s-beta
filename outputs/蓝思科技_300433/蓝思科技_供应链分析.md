# 蓝思科技（300433）供应链分析报告

**主营业务**: AI智能终端精密结构件、功能模组、整机组装（智能手机、AI眼镜、智能汽车、人形机器人、AI服务器等）

## 公司概况

**股票代码**: 300433  
**公司名称**: 蓝思科技股份有限公司  
**上市板块**: 深交所创业板  
**当前价格**: ~28元  
**市值**: ~2941亿元

**业务简介**：
蓝思科技由周群飞于2003年创立，是全球最大的消费电子防护玻璃制造商之一。经过30余年发展，公司已从单一手机玻璃盖板供应商发展为覆盖**智能手机、智能汽车、AI眼镜、人形机器人、AI服务器、商业航天**等多领域的精密制造平台。

公司核心能力在于**脆性材料（玻璃、蓝宝石、陶瓷）的精密后道加工**，以及**金属结构件的精密加工与组装**。2020年收购可胜/可利泰州后，公司进一步整合了金属机壳加工能力，成为少数能提供"玻璃+金属+组装"一站式服务的供应商。

## 供应链关键洞察

### 1. 产业链位置

蓝思科技处于**L2-L3层**——精密玻璃/金属结构件制造与模组组装。

```
L0 资源层:    高纯硅矿（玻璃原料）、铝矿、稀有金属
L1 材料层:    ★康宁/肖特★ — 高铝硅玻璃基材、蓝宝石晶棒
L2 部件层:    ★蓝思科技★ — 玻璃盖板后道加工、金属中框精密加工
L3 制造层:    ★蓝思科技★ — 模组组装、整机组装（EMS）
L4 终端层:    Apple/Samsung/华为/小米 — 品牌商
```

**关键判断**：蓝思科技横跨L2和L3层，但**真正的卡点在L1层**（玻璃基材核心技术由康宁控制）。蓝思科技的后道加工虽然需要深厚工艺积累，但并非不可替代——伯恩光学、比亚迪电子等都具备同等能力。

### 2. 竞争格局

**玻璃盖板**：双雄格局（伯恩光学 vs 蓝思科技），但Apple有意培育第三/第四供应商以分散风险。

**金属结构件**：参与者众多（科森科技、长盈精密、比亚迪电子等），竞争充分。

**整机组装（EMS）**：蓝思科技正向此领域延伸，但面对富士康、立讯精密等EMS巨头，竞争优势不明显。

**新业务（机器人/服务器/航天）**：尚处极早期，竞争格局未定。

### 3. 上下游议价能力

- **对上游**：议价能力中等。玻璃基材依赖康宁等少数供应商（第一大供应商占采购32%），但蓝思体量足够大，有一定谈判筹码
- **对下游**：议价能力弱。Apple作为绝对大客户（45%收入），具有极强的定价权和供应链管理能力，蓝思只能在Apple的价格框架内竞争
- **净效果**：上下游两头受压，毛利率长期维持在13-16%区间

## 风险因素

### 供应链风险
- **核心基材依赖**：高铝硅玻璃基材核心技术在康宁等外企手中，后道加工环节无法建立真正的技术壁垒
- **第一大供应商集中**：占采购32.22%，供应端有一定集中度风险
- **自动化设备进口依赖**：部分高精度CNC、检测设备依赖进口

### 需求侧风险
- **Apple单一客户风险**：45%收入来自Apple，任何供应链调整都将重创公司
- **消费电子周期性**：2026Q1营收-17%已体现行业周期性波动
- **汇率风险**：海外业务占比高，人民币汇率波动直接影响利润（2026Q1汇兑损失为业绩下滑主因之一）

### 其他风险
- **机构大规模撤退**：机构数量从908家（2025Q4）暴降至66家（2026Q1），减幅93%
- **估值风险**：2941亿市值/ROE 2.16%/PE 85倍，严重偏高
- **新业务不确定性**：TGV玻璃基板零收入，人形机器人/服务器业务处于极早期
- **劳动力密集风险**：公司员工数万人，人力成本持续上升

## 结论

蓝思科技（300433）是全球消费电子精密制造领域的重要参与者，在玻璃盖板后道加工领域拥有深厚积累。然而，从**供应链卡点**的角度看：

1. **不是卡点**：真正的卡点在上游玻璃基材（康宁），蓝思科技是后道加工环节，可替代性高
2. **客户集中度致命**：Apple占收入45%，远超Serenity框架40%红线
3. **估值严重偏高**：2941亿市值对应2%ROE，反映的是成长故事而非产业链稀缺性
4. **机构正在撤离**：908→66家机构的暴跌是极其危险的信号

**一句话结论**：蓝思科技是Apple供应链的**关键代工伙伴**，但不是产业链的**卡点持有者**。它是消费电子行业的"富士康"——规模庞大、不可或缺于短期，但可替代于中长期。
