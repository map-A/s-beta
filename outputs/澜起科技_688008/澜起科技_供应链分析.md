## 供应链关键洞察

### 1. 产业链位置

```
L0 DRAM晶圆（三星/SK海力士/美光） → L1 内存模组（RDIMM/LRDIMM） → L2 内存接口芯片（RCD/MDB — 澜起36.8%全球第一） → L3 服务器/AI算力
```

澜起科技位于**L2 互连芯片层**，是连接DRAM存储器与CPU/GPU之间的"翻译官"和"调度中心"。公司产业链位置的独特价值在于：

1. **标准制定权**：DDR5"1+10"架构由公司发明并被JEDEC采纳为国际标准
2. **三寡头格局**：全球CR3=93.4%，公司36.8%排名第一
3. **第二增长曲线**：PCIe Retimer+CXL MXC打开AI运力新空间

**与Serenity全球产业链的对应关系**：
- 公司内存接口芯片 → 对应AI服务器"运力"层（算力+存力+运力三大支柱之一）
- 公司PCIe Retimer → 对应GPU-CPU互连的关键信号调理芯片
- 公司CXL MXC → 对应下一代内存扩展/池化架构的核心控制器

### 2. 竞争格局

**内存接口芯片（全球CR3=93.4%三寡头）**：

| 维度 | 澜起科技 | IDT/瑞萨 | Rambus |
|------|---------|---------|--------|
| 全球市占率 | **36.8%（第一）** | 约30% | 约27% |
| DDR5标准制定 | **牵头制定者** | 参与者 | 参与者 |
| 产品线宽度 | RCD+MDB+CKD+PCIe+CXL+CPU | RCD+MDB | RCD+MDB+PCIe |
| 毛利率 | 65.57%→69.79% | - | ~50% |
| 增速（2025） | +53.43% | - | ~20% |

**PCIe Retimer芯片（全球CR2=96.9%双寡头）**：
- 澜起：10.9%（全球第二，新进入者）
- Rambus/Broadcom（推测）：~86%（全球第一）
- 公司PCIe 5.0/CXL2.0 Retimer是全球主要供货的两家厂商之一

### 3. 上下游议价能力

**对上游**：中等
- Fabless模式依赖台积电代工（集中度高）
- 但公司采购规模大，议价能力逐步增强
- 互动问答暗示台积电Q3起提升产能至150万颗/月

**对下游**：强
- 全球三寡头格局 = 极强定价权（69.79%毛利率证明）
- 客户为全球顶级厂商（三星/SK海力士/美光等）
- 前5大客户占比77.24%——客户集中但均为行业巨头
- DDR5标准制定者 = 客户必须兼容公司产品

## 风险因素

### 供应链风险
- **Fabless依赖**：无自有工厂，依赖台积电代工
- **供应商集中**：前5大供应商占比79.39%
- **地缘政治**：台海风险影响代工供应

### 需求侧风险
- **AI投资周期性**：超大厂Capex下降将冲击服务器出货量
- **客户集中度**：前5大客户占比77.24%
- **DDR5渗透率**：DDR4→DDR5升级节奏可能波动

### 其他风险
- **估值风险**：117倍PE，2984亿市值，已充分定价
- **竞争风险**：Rambus/IDT可能缩小份额差距
- **技术迭代**：DDR6标准制定中，竞争格局可能变化

---

## 结论

澜起科技是A股最具"卡点"属性的半导体设计公司——全球内存接口芯片三寡头之一、DDR5标准牵头制定者、PCIe Retimer全球第二、CXL MXC全球首发。公司69.79%毛利率体现极强定价权，第二增长曲线（PCIe+CXL）打开AI运力新空间。但2984亿市值/117倍PE已充分定价，18家券商全覆盖使信息差为零。建议作为AI运力层核心标的长期跟踪，等待估值回调后介入。
