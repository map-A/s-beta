## 供应链关键洞察

### 1. 产业链位置

长信科技位于显示产业链的中上游材料/模组层（L1-L2），向上采购ITO靶材/玻璃基板等原材料，加工成ITO导电玻璃和触控显示模组后销售给手机/汽车厂商。

**产业链图谱**:
```
L0 资源层      L1 材料层           L2 模组层           L3 终端
┌──────────┐  ┌──────────────┐  ┌──────────────┐  ┌──────────┐
│ 铟锡矿   │→│ ITO靶材     │→│ 长信科技     │→│ 荣耀/OPPO│
│ 石英砂   │  │ 玻璃基板    │  │ (ITO玻璃+   │  │ 比亚迪   │
└──────────┘  └──────────────┘  │  显示模组+   │  │ 其他品牌 │
                                │  算力租赁)   │  └──────────┘
                                └──────────────┘
```

### 2. 竞争格局

**ITO导电玻璃**: 行业参与者众多（长信科技、莱宝高科等10+家），产品同质化，毛利率持续下降（22%→9.5%）。长信是"龙头之一"但不具垄断地位。

**显示模组**: 竞争极度激烈，蓝思科技（2944亿市值/苹果核心供应商）是绝对龙头，京东方/天马等面板巨头也在布局车载显示。

**算力租赁**: 零壁垒业务，大量企业涌入。长信智算刚起步（仅签2.62亿合同），拟投30亿采购服务器——风险巨大。

### 3. 上下游议价能力

**对上游议价**: 弱——ITO靶材原料（铟）为稀缺金属，上游议价权较强
**对下游议价**: 弱——毛利率仅9.5%，下游手机/汽车厂商议价权极强

---

## 风险因素

### 供应链风险
- **铟价波动**: ITO靶材原料铟为稀缺金属，价格波动影响成本
- **上游依赖**: 第一大供应商占比13.08%

### 需求侧风险
- **消费电子周期下行**: 智能手机市场增长放缓，显示模组需求承压
- **竞争加剧**: 毛利率从22%降至9.5%，行业竞争持续恶化
- **算力租赁风险**: 30亿服务器采购将大幅增加负债，算力租赁市场竞争激烈

### 其他风险
- **2025年全年亏损**（-8190万）
- **对外担保39.5亿元**——超过净资产30%
- **历史信披违规**（2019年立案调查）
- **北向资金连续两季度减持40%+**

---

## 结论

长信科技的供应链分析表明：这是一家传统显示材料/模组制造企业，行业充分竞争、毛利率极低（9.5%）、技术壁垒有限。公司通过子公司长信智算切入算力租赁赛道（拟投30亿采购服务器），但这进一步增加了财务风险而非构建护城河。291亿市值/170xPE严重脱离基本面。
