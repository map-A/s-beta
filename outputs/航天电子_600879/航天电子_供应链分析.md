## 供应链关键洞察

### 1. 产业链位置

航天电子处于**航天产业链的系统集成层**：
```
L0 材料层: 特种金属/复合材料 → L1 元器件层: 航天级芯片/继电器/连接器 →
L2 部件层: 惯性导航/遥测/激光通信模块 → L3 系统集成层: 航天电子系统总装 →
L4 终端层: 火箭/卫星/导弹/无人机
```

公司位于**L2-L3层**，同时自产部分L1元器件（芯片/继电器），纵向整合程度较高。

### 2. 竞争格局
- 军工按集团划分，航天科技/航天科工/中航工业各有配套体系
- 在航天配套领域，公司作为航天九院唯一上市平台有独特地位
- 在无人机领域：彩虹(航天彩虹)/翼龙(中航无人机)/飞鸿(航天电子)三足鼎立

### 3. 上下游议价能力
- **对上游**：议价能力中等（军工采购体系内配套为主）
- **对下游**：议价能力弱（军方/航天科技集团为垄断买方，定价受军品定价机制约束）

## 风险因素
