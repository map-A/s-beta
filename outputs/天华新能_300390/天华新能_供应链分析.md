## 供应链关键洞察

### 1. 产业链位置

```
L0 资源层:   锂辉石矿（澳大利亚/南美/江西）
             ↓ 精矿供应（推测供应商A/B为矿山）
L1 加工层 [天华新能/天宜锂业]:
             电池级氢氧化锂 → 高镍NMC正极（CATL/LG）
             电池级碳酸锂 → LFP正极（比亚迪等）
L2 正极材料: 高镍三元正极（容百科技/格林美等）
L3 电芯制造: CATL / LG化学 / SK / 比亚迪（Customer A推测CATL）
L4 整车厂:   特斯拉/比亚迪/大众等
```

**最关键的产业链洞察**：
CATL通过**双重持股**（25%天宜 + 7%天华上市公司）将天华变成了准"集团内供应商"。这是锂盐供应链中极为罕见的安排，与其他独立锂盐供应商相比，天华享有结构性优先采购地位。

### 2. 竞争格局

**全球电池级氢氧化锂市场**：
- Albemarle（美国）：全球最大，但主要服务欧美客户，中国市场份额有限
- 赣锋锂业：中国最大锂盐公司（更综合），但与CATL无同等深度绑定
- 天华新能：**中国CATL体系的首选供应商**，差异化定位

**竞争优势总结**：
- CATL股权绑定 = 定价透明、采购稳定、扩产联动
- 技术积累：高镍正极对LiOH纯度的极致要求，天华已验证多年
- 成本竞争力：四川宜宾 + 甘孜产能布局，水电成本优势

### 3. 上下游议价能力

**对上游（锂矿供应商）**：中等  
- 矿山市场较为分散，天华有一定议价空间
- 积极布局江西自有矿山，未来成本结构改善

**对下游（CATL等电池厂）**：中等偏强  
- CATL作为股东不能任意压价（等于压自己的利润）
- 但整体而言锂盐是商品，价格随LiOH现货市场波动

## 风险因素

### 供应链风险
- **锂辉石矿供应**：主要依赖澳大利亚进口，若矿山国家政策变化或供应中断，原材料受限
- **锂价周期**：2023年跌幅85%的历史教训表明，锂价的周期性波动会直接摧毁利润
- **烧碱等辅料成本**：生产氢氧化锂需大量碱性化学品，若碱价上涨将进一步压缩毛利

### 需求侧风险
- **EV增速放缓**：若中国EV渗透率增速不及预期，锂盐需求将低于当前乐观预测
- **CATL技术路线转变**：若CATL将电池路线从三元（高镍NMC）大规模切换至LFP（LFP不用LiOH），天华高端LiOH需求将受影响
- **固态电池时代**：长期来看固态电池可能改变锂的使用形态，但这是10年以上的风险

### 其他风险
- **单一大客户（CATL 23%）**：尽管CATL是股东，但过度集中仍是结构性弱点
- **H股IPO失败**：若香港市场条件不允许，估值重评机会消失
- **江西矿山整合**：新收购的矿山项目若执行不力，可能不必要地耗费资本

## 结论

天华新能是本次分析中**唯一同时满足Serenity全部4项卡点标准的标的**。其核心价值在于：

1. **战略型卡点**：CATL的双重持股（天宜25%+上市7%）将天华从普通锂盐供应商变成CATL产业链的利益共同体，这是传统Serenity框架"供给集中"标准的创新形式
2. **周期复苏中段**：从Q1 2025的近零利润到Q1 2026的EPS 1.166，正处于锂价复苏上行通道中，但PE仍仅15x，远低于同类赣锋/天齐
3. **多重催化剂**：HK IPO进行中+创业板指新纳入+江西矿山布局
4. **防静电超净业务**（美光/AMD/中芯客户）提供下行时的缓冲基本盘

建议**核心仓位3-5%**，是本次分析中最优先的买入标的。
