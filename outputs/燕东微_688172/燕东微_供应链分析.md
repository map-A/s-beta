## 供应链关键洞察

### 1. 产业链位置

燕东微位于半导体产业链的**L3制造层**，采用IDM（垂直整合制造）模式，同时开展自有品牌产品和晶圆代工业务：

```
L0 资源层：硅料/碳化硅衬底 → 
L1 设备材料层：半导体设备/光刻胶/特气 → 
L2 核心部件层：硅片/外延片 → 
L3 制造层：【燕东微】6/8/12英寸晶圆制造（IDM+代工）→ 
L4 终端层：新能源车/光伏/消费电子/硅光通信
```

公司独特的差异化布局：
- **6英寸产线**：成熟产品（分立器件、模拟IC），年产69万片
- **8英寸产线**：功率器件+BCD平台+硅光SiN平台，年产63万片
- **12英寸产线**：燕东科技（月产2万片，目标4万片）+ 北电集成（在建，目标5万片）
- **硅光平台**：8英寸SiN（已量产，年产1.48万片）+ 12英寸SOI（PDK发布，客户导入中）
- **第三代半导体**：6英寸SiC SBD工艺平台已量产

### 2. 竞争格局

**功率半导体IDM**：
- 国内：华润微（营收~114亿）> 士兰微（~93亿）> 扬杰科技（~60亿）> 华微电子（~30亿）> 燕东微（~18亿）
- 全球：英飞凌、安森美、ST、罗姆等国际巨头主导
- 燕东微在国内IDM企业中体量较小，但拥有12英寸产线布局的差异化优势

**特色工艺代工**：
- 华虹半导体为国内龙头（营收~200亿）
- 中芯国际（成熟制程）、晶合集成等也有布局
- 燕东微代工业务处于起步阶段

**硅光平台**：
- 全球硅光子市场2025年约26.5亿美元，2030年预计96.5亿美元
- 国内参与者少，燕东微的8英寸SiN+12英寸SOI布局较为领先
- 但商业化规模极小，尚未形成实质性收入

### 3. 上下游议价能力

**对上游议价**：弱
- 半导体设备（尤其12英寸线设备）议价权在供应商手中
- 进口设备受地缘政治影响，供应链不确定性大
- 材料端国产替代推进中，但核心材料仍依赖进口

**对下游议价**：弱
- 功率半导体市场竞争充分，定价权分散
- 公司体量较小（18亿营收），面对大客户议价能力有限
- 代工业务价格受华虹半导体等大厂压制

## 风险因素

### 供应链风险
- **设备进口依赖**：12英寸产线核心设备（光刻机等）主要依赖进口，受美国出口管制影响
- **材料国产替代不完整**：SOI衬底、碳化硅衬底等核心材料仍部分依赖进口
- **产能爬坡不确定性**：两条12英寸线同时运营，管理复杂度高

### 需求侧风险
- **功率半导体行业周期性**：分立器件2025年全球销售额同比下降0.4%
- **价格竞争加剧**：国内功率半导体产能快速扩张，可能面临价格战
- **终端需求波动**：新能源车、光伏等下游行业增速放缓风险

### 其他风险
- **持续深度亏损**：2025年毛利率-17.92%，净亏损4.08亿元
- **巨额资本开支**：12英寸产线投资巨大，折旧压力持续
- **国家大基金减持**：一期已开始减持，市场情绪承压
- **估值泡沫**：1203亿市值/亏损企业，PS 65倍

---

## 结论

燕东微是国内半导体行业中少数拥有6/8/12英寸完整产线的IDM企业，具备硅光平台和碳化硅第三代半导体的前沿布局。公司的差异化在于：

1. **12英寸产线**：国内IDM企业中布局较早，燕东科技已实现量产（月产2万片）
2. **硅光平台**：8英寸SiN+12英寸SOI，国内硅光子制造领域稀缺资源
3. **第三代半导体**：SiC SBD工艺平台已量产
4. **政策支持**：国家大基金一/二期双重持股

但当前面临的核心问题是：
- **1203亿市值对应亏损企业**，估值严重偏离基本面
- **2025年毛利率-17.92%**，研发投入占比43.74%，短期内盈利无望
- **功率半导体市场充分竞争**，不具备卡点级别的定价权

公司处于"技术布局有价值、但当前价格不值得"的状态。建议等待12英寸产线达到设计产能、毛利率转正、估值回调至合理区间后再考虑介入。
