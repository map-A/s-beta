# 杭电股份（603618）供应链分析报告

**生成日期**: 2026-06-25  
**所属行业**: 电力设备 / 电网设备 + 光通信  
**主营业务**: 电力电缆、光通信（光纤光缆）、铜箔

---

## 公司概况

**股票代码**: 603618  
**公司名称**: 杭州电缆股份有限公司  
**上市板块**: 上交所主板  
**当前价格**: 28.49元  
**市值**: ~395亿元

**业务简介**：
杭电股份前身杭州电缆厂成立于上世纪60年代，是国内电线电缆行业的老牌企业之一。公司目前拥有三大业务板块：
1. **电力电缆**（核心业务，占营收95%以上）：产品涵盖500kV及以下高压超高压交联电力电缆、特高压导线、民用线等
2. **光通信**（占比约4.6%）：通过子公司永特信息生产光纤预制棒和光纤，年产150吨光纤预制棒暨500万芯公里光纤项目
3. **铜箔**（新业务）：子公司江西杭电铜箔投资50亿元建设年产5万吨新能源汽车锂电池超薄铜箔项目

## 供应链关键洞察

### 1. 产业链位置

杭电股份处于**L3制造集成层**——它不是上游稀缺原材料或设备的提供商，而是将大宗商品（铜、铝、化工材料）加工为标准化电力设备（电缆、导线、光缆）的中游制造商。

```
L0 资源层:    铜矿/铝矿、硅材料、化工原料
L1 基础材料:   电解铜、电解铝、聚乙烯、光纤预制棒
L2 核心部件:   铜导体、光纤、绝缘材料
L3 制造层:     ★杭电股份★ — 电力电缆、光缆、铜箔
L4 终端应用:   电网建设、5G/AI数据中心、新能源汽车
```

**关键判断**：杭电股份所处的L3层是**竞争最激烈、毛利最薄**的层级。真正有定价权的环节在上游（铜矿、光纤预制棒）或终端（国家电网作为唯一大买家的议价权）。

### 2. 竞争格局

**电力电缆**：行业极度分散（CR10<15%），4000+企业参与。杭电股份的优势在于品牌（"永通"驰名商标）和华东区域渠道，但全国范围内面临远东、宝胜、中天等强劲竞争。

**光纤光缆**：行业集中度高（光棒CR4约85%），但杭电股份是**边缘参与者**。光纤光缆业务仅占公司总收入4.6%，且规模远小于长飞、亨通、中天。2025年光纤价格低谷时，该板块毛利率为负。

**铜箔**：新进入者，项目仍在建设/爬坡期，尚未形成规模竞争力。复合铜箔行业处于0到1阶段，竞争格局未定。

### 3. 上下游议价能力

- **对上游**：议价能力弱。铜铝等大宗商品是全球定价，公司只能被动接受
- **对下游**：议价能力弱。国家电网/南方电网作为唯一大买家，通过招标压价，电缆企业只能在招标框架内竞争
- **净效果**：毛利率被上下游两头压缩，长期维持在7-13%区间

## 风险因素

### 供应链风险
- **铜价波动**：铜价每上涨10%，公司毛利率将下降约3-5个百分点（因铜材占成本70%+）
- **原材料供应集中**：前5大供应商采购占比37.51%，供应端有一定集中度
- **光纤预制棒产能爬坡**：子公司永特信息的光棒项目从小批量到规模化生产，仍存在工艺优化和技术迭代风险

### 需求侧风险
- **国网投资节奏**：电力电缆需求高度依赖国家电网/南方电网招标，若"十五五"投资不及预期（4万亿规划可能滞后），将直接冲击公司营收
- **光纤集采价格**：运营商集采价格是光纤光缆行业的"定价锚"，若2026年集采价格涨幅不及预期，光通信板块利润弹性将大打折扣
- **铜箔商业化进度**：复合铜箔行业处于早期验证阶段，若下游电池厂导入不及预期，50亿元投资将面临回报周期拉长风险

### 其他风险
- **大股东减持**：富春江通信集团（持股21.5%）已公告减持计划（不超过3%），6月已完成部分减持（均价40.42元）
- **估值风险**：395亿市值对应2025年净亏损，即使2026年盈利预测兑现，PE仍高于行业均值
- **杠杆风险**：公司2025年资产负债率74.3%，处于较高水平；融资余额12.34亿元（杠杆资金参与度高）
- **汇率风险**：公司有少量境外业务（占比2.3%），人民币汇率波动影响有限但存在

## 结论

杭电股份（603618）是一家典型的**中游电力设备制造商**，正受益于光纤价格上涨的周期性机遇和国家电网大规模投资的政策红利。但其在产业链中的位置决定了它**不具备卡点属性**——上游原材料被大宗商品定价主导，下游被电网/运营商的招标议价权压制，行业竞争格局极度分散。

从供应链角度看，公司当前395亿市值已经充分反映了光纤涨价和国网投资的乐观预期。如果以"卡点投资"思维分析，真正值得关注的是：上游光纤预制棒的集中度（长飞/亨通/中天/富通CR4约85%）以及AI算力对光纤互联需求的结构性增长——但这些机会对应的标的是**长飞光纤、亨通光电**等龙头，而非杭电股份。

**一句话结论**：杭电股份是光纤概念的**受益者**，但不是产业链的**瓶颈持有者**。
