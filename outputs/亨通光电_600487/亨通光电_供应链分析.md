## 供应链关键洞察

### 1. 产业链位置
亨通光电处于**L1-L2光纤光缆/海缆制造层**，是光棒→光纤→光缆→海底光缆的全产业链企业。

**产业链定位**：
```
上游：高纯石英 + 铜铝 + 设备
  ↓
中游：亨通光电（光棒→光纤→光缆→海底光缆/电缆）
  ↓
下游：电信运营商 + 电网 + 海上风电 + 数据中心
```

**关键特征**：
- 全产业链布局（光棒→光纤→光缆→海缆），成本优势
- 但光纤/光缆是大宗商品化产品，毛利率仅10-16%
- 海缆壁垒相对较高（码头、船舶、资质），但非垄断

### 2. 竞争格局
- **光纤光缆**：长飞/亨通/中天三家寡头，但产能过剩+价格竞争激烈
- **海缆**：中天/亨通/东方电缆三家竞争，壁垒相对较高
- **光棒**：长飞是龙头，亨通/中天跟随

### 3. 上下游议价能力
- **对上游**：议价能力中等（采购规模大但原材料价格波动大）
- **对下游**：议价能力弱（运营商/电网强势，光纤价格受招标压制）
- **毛利率趋势**：从2025年的9.89% → 2026Q1的15.98%——改善明显

## 风险因素

### 供应链风险
- 原材料（铜铝/石英）价格波动大
- 光纤光缆产能过剩导致价格竞争

### 需求侧风险
- 电信运营商投资周期性波动
- 海上风电政策调整风险
- 光纤光缆招标价格持续下行

### 其他风险
- 估值风险：PE 82x，股价已涨368%
- 海外经营风险
- 财务风险：负债率51.88%

## 结论

亨通光电是**国内光纤光缆龙头+海缆前三**，但**不是Serenity卡点**：

1. 光纤光缆是大宗商品化产品——毛利率仅10%
2. 供给分散——长飞/亨通/中天三家竞争+产能过剩
3. 估值已充分定价——2,651亿市值，PE 82x
4. 北向资金减持——Smart Money获利了结

**投资建议**：⚠️ 观察等待。等待回调至60-70元+海缆业务放量确认后再考虑。
