## 供应链关键洞察

### 1. 产业链位置

中天科技位于**光通信+海上风电两条产业链的核心设备层**，具备罕见的"全产业链覆盖"能力：

```
光通信链：
光纤预制棒原材料 → 【中天科技：预制棒→光纤→光缆→光模块】→ 数据中心/运营商
                                        ↑
                              AI算力互联核心供应链

海上风电链：
铜材/铠装材料 → 【中天科技：海底电缆制造+敷设工程】→ 海上风电场
                              ↑
                      在手订单121亿元

电力链：
铜材/绝缘材料 → 【中天科技：电力电缆/输配电设备】→ 电网/工业用户
```

公司最独特的竞争优势是**"棒纤缆一体化"**（光纤预制棒→光纤→光缆全链自主）和**"输配一体化"**（电力电缆+输配电设备），这种全产业链布局在行业内较为罕见。

### 2. 竞争格局

**光纤光缆领域**（竞争偏激烈）：
- 国内主要竞争者：亨通光电、长飞光纤、烽火通信、富通集团
- 市场从"价格战"转向"量价齐升"，行业集中度在提升
- 中天科技在特种光缆/特种导线领域品牌最强

**海底电缆领域**（竞争较集中）：
- 国内主要竞争者：东方电缆、亨通光电（3-4家核心企业）
- 高压/深远海海缆技术壁垒极高
- 中天科技是国内最早大规模进入海缆领域的企业，技术积累最深
- 多项深远海海缆技术打破国内空白

**800G光模块领域**（竞争激烈）：
- 国内主要竞争者：中际旭创、新易盛、光迅科技
- 已完成800G批量出货，掌握COB封装能力
- 正向1.6T光模块演进

### 3. 上下游议价能力

**对上游**：议价能力较强
- 自主掌握预制棒技术，关键原材料自主可控
- 大宗采购（铜材等）有规模优势

**对下游**：议价能力中等
- 运营商/电网公司议价权强（集中采购）
- 海缆客户（风电开发商）因技术壁垒中天有一定议价权
- AI算力客户对800G光模块需求急迫，议价权暂可
- 前5大客户合计占收入30.05%，最大客户15.02%

## 结论

中天科技是中国光通信+海缆行业的"双龙头"，具备全产业链覆盖的独特优势。当前正处于"光纤量价齐升+海缆订单饱满+800G光模块放量"的三重增长拐点，2026Q1利润+46%已验证。

**供应链核心观察**：
1. **光纤景气拐点已至**：供给偏紧+散纤涨价+AI需求高增，Q2/Q3利润弹性将进一步释放
2. **海缆在手订单121亿元**：2-3年业绩确定性高，深远海技术壁垒持续构建
3. **800G光模块批量出货**：从光纤延伸到光模块，切入AI算力核心供应链
4. **北向资金大幅增持**：Q1增持62.7%，UBS/CITIGROUP新进QFII
5. **估值已不便宜**：2324亿市值、72x PE，需业绩高增长兑现支撑

**投资含义**：中天科技基本面扎实，增长逻辑清晰，但当前68元已充分定价。等待光纤涨价周期放缓导致的回调（50-55元区间），再考虑左侧建仓。
